---

> https://docs.github.com/en/copilot/using-github-copilot/guides-on-using-github-copilot/writing-tests-with-github-copilot
1. [x] It generates a suite of unit tests for the currently open file, using context from existing test files if available
1. [ ] It runs all existing unit tests in the project and displays the results in the chat window
1. [ ] It analyzes existing test coverage and suggests which areas of code need additional tests
1. [ ] It converts existing tests from one testing framework to another
