question: "What is an alias used for when using GitHub Copilot CLI?"
---

> https://learn.microsoft.com/en-us/training/modules/github-copilot-across-environments/4-git-hub-copilot-for-the-command-line
1. [x] Using an alias enables GitHub Copilot CLI to automatically execute commands in the command line.
1. [ ] Using an alias creates custom shortcuts for frequently used Copilot Chat prompts.
1. [ ] Using an alias allows you to switch between different AI models in the CLI.
1. [ ] Using an alias enables Copilot to remember your command history across terminal sessions.
