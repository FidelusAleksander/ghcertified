title: "Question 014"
question: "What is the easiest way to get started with GitHub Copilot?"
---

> https://docs.github.com/en/copilot/using-github-copilot/getting-started-with-github-copilot
1. [ ] Configure GitHub Copilot through the command line interface using `gh copilot setup`.
1. [ ] Enable Copilot in your GitHub account settings and it will automatically work in all development environments.
1. [x] Install the Copilot extension in your preferred environment, such as Visual Studio Code.
1. [ ] Clone the GitHub Copilot repository and build the extension from source.
