- [x] オプションの使用状況分析はオプトアウト可能である。
- [x] Copilot CLIが生成した応答は、各提案後に確認できる。
- [ ] プロンプトと使用状況分析の両方がデフォルトで保持される。
> プロンプトはデフォルトで保持されません。
- [ ] 匿名化されているためオプションの使用状況分析をオプトアウトすることはできない。
- [ ] GitHub Copilot CLIはコマンドの説明や提案しかできず、ユーザーに代わって実行することはできない。
