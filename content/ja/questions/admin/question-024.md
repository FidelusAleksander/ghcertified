- [x] ユーザー名とパスワード（オプションで2FA）
- [x] Personal access token（PAT）
- [x] SSH KEY
- [x] Enterpriseアカウント向けのSAML SSO
- [x] パスキー認証（パスワードレスログインのオプトインβ版）
- [x] サードパーティアプリ統合用のOAuth Token
- [ ] モバイルデバイスにリンクされたQRコードシステム
- [ ] FacebookやGoogleなどのSNSアカウントによる認証
