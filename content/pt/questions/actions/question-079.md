---
title: "Pergunta 079"
question: "Em qual commit e branch os fluxos de trabalho agendados são executados no GitHub Actions?"
---

> https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#schedule

1. [ ] Os fluxos de trabalho agendados são executados em um commit específico na branch modificada por último.
   > incorreto, tanto o commit específico quanto a branch modificada por último
1. [ ] Os fluxos de trabalho agendados são executados em um commit específico na branch main.
   > incorreto, tanto o commit específico quanto a branch main
1. [x] Os fluxos de trabalho agendados são executados no commit mais recente na branch padrão do repositório.
1. [ ] Os fluxos de trabalho agendados são executados no commit mais recente na branch main.
   > o commit mais recente está correto, mas a branch main não
