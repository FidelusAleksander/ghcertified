---
title: "Pergunta 094"
question: "O que são pull requests de rascunho?"
---

> https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests#draft-pull-requests
1. [x] Pull requests de rascunho são pull requests que ainda não estão prontos para revisão.
1. [ ] Pull requests de rascunho são pull requests que já foram revisados, mas não foram mergeados.
1. [ ] Pull requests de rascunho são pull requests que já foram revisados, mas não foram aprovados.
1. [ ] Pull requests de rascunho são templates para criar novos pull requests.
