---
title: "Questão 020"
question: "Fluxos de trabalho do GitHub Actions são acionados por eventos. Quais destes são eventos válidos suportados pelo GitHub Actions? (Escolha dois.)"
---



> https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#about-events-that-trigger-workflows
- [x] Um commit é feito em uma branch
- [x] Um pull request é aberto
- [ ] Uma alteração é feita nas configurações do repositório
- [ ] Uma nova vulnerabilidade foi detectada em uma dependência
> Este não é um evento válido que poderia acionar um fluxo de trabalho. É uma funcionalidade do Dependabot.
- [ ] Um novo membro foi adicionado ao repositório
