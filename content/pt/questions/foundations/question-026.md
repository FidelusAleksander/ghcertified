question: "O que é um GitHub Codespace?"
---


> https://docs.github.com/en/codespaces/overview
1. [x] É um ambiente de desenvolvimento pré-configurado especificamente configurado para um repositório. Ele permite que você comece imediatamente a escrever código para um projeto sem precisar configurar um ambiente de desenvolvimento local.
1. [ ] Um GitHub Codespace é um ambiente de codificação holográfico interativo que requer hardware especial para ser utilizado.
1. [ ] GitHub Codespace é uma ferramenta de criação de código da plataforma de IA que gera automaticamente código quase perfeito.
1. [ ] GitHub Codespace é um evento anual hospedado pelo GitHub onde desenvolvedores podem se reunir e discutir tópicos sobre codificação.
