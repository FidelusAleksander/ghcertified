title: "Questão 088"
question: "Como você pode vincular um pull request a um problema? (Escolha duas opções.)"
---

> https://docs.github.com/en/issues/tracking-your-work-with-issues/linking-a-pull-request-to-an-issue
- [x] Manualmente, seja na visualização do pull request ou do problema.
- [x] Usando uma palavra-chave na descrição do pull request e referenciando o número do problema.
- [ ] Usando uma palavra-chave na descrição do pull request e referenciando o título do problema.
- [ ] Incluindo o número do problema nas alterações de código.
- [ ] Incluindo o número do problema na mensagem do commit.
