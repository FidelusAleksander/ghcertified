---
title: "Pergunta 008"
question: "Quais desses fluxos de trabalho são automações incorporadas no GitHub Projects? (Escolha dois.)"
---


> https://docs.github.com/pt/issues/planning-and-tracking-with-projects/automating-your-project/using-the-built-in-automations
- [x] Quando problemas ou pull requests no seu projeto são encerrados, o status deles é definido como Concluído.
- [x] Quando pull requests no seu projeto são mesclados, o status deles é definido como Concluído.
- [ ] Quando problemas ou pull requests são abertos no seu projeto, o status deles é definido como Concluído.
- [ ] Quando um novo colaborador é adicionado a um repositório, os problemas não atribuídos são atribuídos a ele.
- [ ] Quando uma GitHub Action é acionada, criar um novo item no seu GitHub Project.
