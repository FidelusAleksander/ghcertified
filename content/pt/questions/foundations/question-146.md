---
title: "Pergunta 146"
question: "Qual é a diferença entre Git e GitHub?"
---

> **Mais informações**: [Git vs GitHub](https://learn.microsoft.com/en-us/training/modules/intro-to-git/1-what-is-vc)
1. [x] Git permite que você trabalhe com um ou mais branches locais e envie alterações para um remote repository. GitHub atua como o remote repository, que é acessado através de um site ou ferramentas de linha de comando.
1. [ ] Git é um sistema de controle de versão distribuído (DVCS) que funciona na nuvem. GitHub é uma camada de interface que fornece acesso à tecnologia Git.
1. [ ] Git é usado por um contribuinte individual. GitHub é usado por vários contribuidores para simplificar o trabalho de desenvolvimento em grupo.
