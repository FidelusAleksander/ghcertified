title: "Pergunta 013"
question: "Como você pode atribuir uma pessoa a um problema ou pull request?"
---


> https://docs.github.com/pt/issues/tracking-your-work-with-issues/assigning-issues-and-pull-requests-to-other-github-users
1. [x] Usando o campo Responsáveis na barra lateral
1. [ ] Adicionando uma etiqueta com o nome de usuário deles do GitHub ao problema ou pull request
1. [ ] Mencionando-os na descrição do problema ou pull request
1. [ ] Não é possível atribuir uma pessoa a um problema ou pull request.
