---
title: "Pergunta 010"
question: "Qual é a diferença entre GitHub projects e GitHub projects classic?"
---

> https://docs.github.com/en/issues/organizing-your-work-with-project-boards/managing-project-boards/about-project-boards
1. [x] GitHub Projects é a nova experiência do GitHub que oferece diversas novas funcionalidades e melhorias em relação ao GitHub Projects classic mais antigo.
1. [ ] GitHub Projects classic é a nova experiência do GitHub que oferece diversas novas funcionalidades e melhorias em relação ao GitHub Projects mais antigo.
1. [ ] GitHub Projects classic é uma forma mais simples de gerenciar seus GitHub Projects e é recomendado para iniciantes.
1. [ ] GitHub Projects classic é a versão aprimorada que, além do GitHub Projects, oferece alguns templates pré-definidos como roadmaps e sprints.
