question: "Jak można wykluczyć określone pliki z GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/how-tos/configure-content-exclusion/exclude-content-from-copilot"

- [ ] Edytowanie pliku .gitignore  
> .gitignore jest używany do wykluczania plików z Git, a nie z Copilot  
- [x] Przejście do ustawień repozytorium na GitHub i dodanie ścieżek do wykluczenia  
- [ ] Konfigurowanie wykluczeń w pliku konfiguracyjnym Copilot  
- [ ] Używanie polecenia w terminalu
