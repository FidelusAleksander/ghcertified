---
question: "Jak możesz dostarczyć GitHub Copilot kontekst, aby generował dostosowane odpowiedzi dla Twojego repozytorium?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/copilot/customizing-copilot/adding-custom-instructions-for-github-copilot"
---

- [x] Poprzez utworzenie pliku o nazwie `.github/copilot-instructions.md` w repozytorium
- [ ] Poprzez wysłanie e-maila do wsparcia GitHub z szczegółami projektu
- [ ] Poprzez zmodyfikowanie pliku `.gitconfig`, aby zawierał niestandardowe instrukcje
> Modyfikacja pliku `.gitconfig` nie zapewnia niestandardowych instrukcji dla GitHub Copilot.
- [ ] Poprzez utworzenie GitHub issue o nazwie `copilot-instructions` w repozytorium z odpowiednim kontekstem
> Utworzenie GitHub issue nie dostarcza niestandardowych instrukcji dla GitHub Copilot.
