---
question: "W jaki sposób możesz dostarczyć GitHub Copilot kontekst, aby generował dostosowane odpowiedzi dla twojego repository?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/copilot/how-tos/copilot-on-github/customize-copilot/add-custom-instructions/add-repository-instructions"
---

- [x] Tworząc plik o nazwie `.github/copilot-instructions.md` w repository
- [ ] Wysyłając e-mail do wsparcia GitHub z szczegółami projektu
- [ ] Modyfikując plik `.gitconfig`, aby dodać niestandardowe instrukcje
> Modyfikacja pliku `.gitconfig` nie dostarcza niestandardowych instrukcji do GitHub Copilot.
- [ ] Tworząc Issue na GitHub o nazwie `copilot-instructions` w repository z niezbędnym kontekstem
> Tworzenie Issue na GitHub nie dostarcza niestandardowych instrukcji do GitHub Copilot.
