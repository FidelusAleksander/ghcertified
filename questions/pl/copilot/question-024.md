---
question: "Jak działa funkcja dopasowywania publicznego kodu w GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/how-tos/get-code-suggestions/find-matching-code"
---

- [x] Wyszukuje dopasowania, porównując sugestie kodu z indeksem publicznych repozytoriów GitHub, który jest odświeżany co kilka miesięcy
- [ ] Wykonuje wyszukiwania w czasie rzeczywistym we wszystkich repozytoriach GitHub, w tym prywatnych
- [ ] Dopasowuje tylko kod z repozytoriów utworzonych w ciągu ostatnich 24 godzin
- [ ] Sprawdza kod w zewnętrznych platformach hostowania kodu poza GitHub
