---
question: "Czym są inteligentne akcje?"
documentation: "https://learn.microsoft.com/en-us/training/modules/get-started-github-copilot/11-examine-inline-chat-smart-actions-quick-chat"
---

- [x] Inteligentne akcje to typowe i predefiniowane zadania, takie jak wyjaśnianie kodu, poprawianie kodu czy generowanie testów i dokumentacji, które mogą być wykonywane przez GitHub Copilot Chat bez konieczności tworzenia dla nich zapytań.
- [ ] Inteligentne akcje to zaawansowana modalność GitHub Copilot używana do debugowania i poprawiania nieudanych testów.
- [ ] Inteligentne akcje to predefiniowane zadania zaprojektowane w celu wspierania typowych zadań programistycznych wewnątrz organizacji, które mogą być ustawiane na poziomie repository i organization.
