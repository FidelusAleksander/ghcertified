question: "Które stwierdzenie poprawnie opisuje GitHub Copilot CLI?"
documentation: "https://docs.github.com/en/copilot/concepts/agents/copilot-cli/about-copilot-cli"
---

- [x] Umożliwia korzystanie z Copilot z poziomu terminala do zadawania pytań, pisania i debugowania kodu oraz interakcji z GitHub.com
- [ ] Jest ograniczone do generowania aliasów shell dla `gh copilot suggest` i `gh copilot explain`
- [ ] Działa tylko w interfejsie internetowym GitHub i nie może uzyskać dostępu do lokalnych plików projektu
- [ ] Automatycznie wykonuje każdy zasugerowany polecenie shell bez pytania o zgodę
