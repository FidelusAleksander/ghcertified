- [x] Użycie aliasu umożliwia GitHub Copilot CLI automatyczne wykonywanie poleceń w wierszu poleceń.
- [ ] Użycie aliasu pozwala wskazać w commitach Git, że zmiany zostały stworzone przez GitHub Copilot.
- [ ] Użycie aliasu pozwala GitHub na przeglądanie i tworzenie pull requestów w Twoim imieniu.
