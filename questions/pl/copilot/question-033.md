- [x] Użycie aliasu pozwala GitHub Copilot CLI automatycznie wykonywać polecenia w wierszu poleceń.
- [ ] Użycie aliasu pozwala wskazać w commitach Git, że zmiany zostały wykonane przez GitHub Copilot.
- [ ] Użycie aliasu umożliwia GitHub przeglądanie i tworzenie pull requestów w Twoim imieniu.
