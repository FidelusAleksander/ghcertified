---
question: "Co jest wymagane do pobrania lub publikowania pakietu GitHub w ramach workflow, takich jak GitHub Actions lub inne narzędzia CI/CD?"
documentation: "https://docs.github.com/en/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages"
---

- [x] Użyj `GITHUB_TOKEN` do publikowania pakietów powiązanych z repozytorium workflow oraz PAT do instalowania pakietów z prywatnych repozytoriów.
- [ ] GitHub Packages nie mogą być używane w workflow GitHub Actions ani w żadnych narzędziach CI/CD.
- [ ] Tylko klucze SSH są wymagane do korzystania z GitHub Packages w workflow.
- [ ] Ręczne pobieranie i przesyłanie pakietów jest wymagane dla każdego uruchomienia workflow.
