- [x] Centralne zarządzanie tożsamością i dostępem członków przedsiębiorstwa na GitHub za pomocą dostawcy tożsamości (IdP).
- [ ] Umożliwienie użytkownikom samodzielnego zarządzania informacjami profilowymi i dostępem do repozytoriów.
- [ ] Umożliwienie użytkownikom tworzenia publicznej zawartości i współpracy z innymi użytkownikami na GitHub.
- [ ] Zapewnienie platformy dla indywidualnych programistów do zarządzania swoimi prywatnymi projektami.
