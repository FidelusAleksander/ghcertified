---
question: "Jakie uprawnienia posiada menedżer GitHub App w ramach organizacji?"
documentation: "https://docs.github.com/en/organizations/managing-programmatic-access-to-your-organization/adding-and-removing-github-app-managers-in-your-organization#about-github-app-managers"
---

- [x] Menedżerowie GitHub App mogą zarządzać ustawieniami rejestracji aplikacji GitHub App należących do organizacji, ale nie mają uprawnień do instalowania ani odinstalowywania aplikacji GitHub App.
- [ ] Mogą uzyskiwać dostęp do kodu źródłowego i go modyfikować dla dowolnych aplikacji GitHub App należących do organizacji.
- [ ] Menedżerowie GitHub App są jedynymi rolami, które mogą tworzyć nowe aplikacje GitHub App w organizacji.
- [ ] Mają możliwość przyznawania uprawnień do instalacji aplikacji zewnętrznym współpracownikom.
