question: "Jak synchronizacja zespołów z Microsoft Entra ID (wcześniej Azure AD) przynosi korzyści organizacjom GitHub Enterprise Cloud?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/managing-iam/using-saml-for-enterprise-iam/managing-team-synchronization-for-organizations-in-your-enterprise"
---

- [x] Umożliwia automatyczne odzwierciedlenie zmian w członkostwie grupy IdP na GitHub, redukując potrzebę ręcznych aktualizacji i niestandardowych skryptów poprzez synchronizację zespołów w organizacjach GitHub z grupami IdP.
- [ ] Umożliwia GitHub pełnienie roli dostawcy tożsamości dla Microsoft Entra ID, upraszczając procesy uwierzytelniania użytkowników.
- [ ] Automatycznie konwertuje członkostwo w grupach Microsoft Entra ID na odpowiadające im repozytoria GitHub, ułatwiając przypisywanie zasobów.
- [ ] Tworzy kopie zapasowe danych organizacji GitHub na serwerach Microsoft Entra ID, zapewniając dodatkową warstwę redundancji danych.
