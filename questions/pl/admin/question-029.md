---
question: "Jakie jest główne ograniczenie kont użytkowników zarządzanych przez Enterprise w GitHub Enterprise Cloud?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/overview/about-github-enterprise-cloud"
---

- [x] Nie mogą wchodzić w interakcję z publicznymi repozytoriami poza ich przedsiębiorstwem.
- [ ] Są ograniczeni do jednej organizacji w ramach konta GitHub Enterprise.
- [ ] Wymagają zgody administratora na tworzenie lub dołączanie do organizacji poza przedsiębiorstwem.
- [ ] Dostęp do integracji zewnętrznych jest ograniczony i musi zostać zatwierdzony przez administratora przedsiębiorstwa.
