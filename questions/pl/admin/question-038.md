question: "Jak właściciel organizacji może znaleźć statystyki dotyczące wykorzystania licencji dla GitHub Enterprise Server?"
documentation: https://docs.github.com/en/enterprise-server@3.21/billing/how-tos/manage-plan-and-licenses/view-enterprise-usage
---

- [x] Przechodząc do panelu `Site admin`, aby sprawdzić wykorzystanie licencji w ustawieniach konta `Enterprise`.
- [ ] Poprzez bezpośrednie prośby e-mail do GitHub Support.
- [ ] Sprawdzając publiczny profil organizacji w celu uzyskania statystyk w czasie rzeczywistym.
- [ ] Korzystając z GitHub API, aby pobrać dane o wykorzystaniu licencji w czasie rzeczywistym dla GitHub Enterprise Server.
