---
question: "Czym są polityki przedsiębiorstwa w kontekście GitHub Enterprise Cloud?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/enforcing-policies/enforcing-policies-for-your-enterprise/enforcing-repository-management-policies-in-your-enterprise"
---

- [x] Polityki przedsiębiorstwa narzucają jednolitą politykę dla wszystkich organizacji w ramach przedsiębiorstwa.
- [ ] Indywidualne ustawienia użytkownika, które dotyczą tylko kont osobistych w GitHub Enterprise Cloud, niezwiązane z zarządzaniem organizacyjnym lub przedsiębiorstwem.
- [ ] Zbiór zalecanych praktyk dla projektów open-source hostowanych na GitHub, mających na celu zwiększenie zaangażowania społeczności i kontrybucji.
- [ ] Protokoły bezpieczeństwa, które automatycznie skanują i naprawiają podatności w bazach kodu we wszystkich repozytoriach w ramach konta przedsiębiorstwa.
