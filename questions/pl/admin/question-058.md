---
question: "Jaką rolę pełni Security Manager w organizacji GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] Security Manager ma uprawnienia do przeglądania alertów bezpieczeństwa i zarządzania ustawieniami dotyczącymi bezpieczeństwa kodu w całej organizacji, a także do odczytu wszystkich repozytoriów.
- [ ] Security Manager jest odpowiedzialny za transakcje finansowe i bezpieczeństwo informacji rozliczeniowych w organizacji.
- [ ] Może usuwać lub przenosić własność dowolnego repozytorium w organizacji ze względów bezpieczeństwa.
- [ ] Security Manager wyłącznie zarządza ustawieniami uwierzytelniania dwuskładnikowego dla wszystkich członków organizacji.
