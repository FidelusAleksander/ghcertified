---
question: "Jaka jest różnica między aplikacjami GitHub Apps a aplikacjami OAuth?"
documentation: "https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/differences-between-github-apps-and-oauth-apps"
---

- [x] GitHub Apps oferują szczegółowe uprawnienia do konkretnych repozytoriów, podczas gdy aplikacje OAuth Apps żądają dostępu do danych użytkownika we wszystkich repozytoriach.
- [ ] GitHub Apps i OAuth Apps nie różnią się; zapewniają ten sam poziom uprawnień.
- [ ] OAuth Apps oferują bardziej szczegółowe uprawnienia w porównaniu do GitHub Apps.
- [ ] GitHub Apps mogą uzyskiwać dostęp jedynie do publicznych repozytoriów, podczas gdy OAuth Apps mogą uzyskiwać dostęp zarówno do repozytoriów publicznych, jak i prywatnych.
