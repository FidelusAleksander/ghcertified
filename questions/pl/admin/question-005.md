---
question: "Które z poniższych są poprawnymi metodami generowania i udostępniania pliku diagnostycznego dla GitHub Enterprise Server?"
documentation: "https://docs.github.com/en/enterprise-server/support/contacting-github-support/providing-data-to-github-support#creating-and-sharing-diagnostic-files"
---

- [x] Użyj narzędzia wiersza poleceń `ghe-diagnostics` do pobrania diagnostyki dla swojej instancji.
- [x] Utwórz plik diagnostyczny w Konsoli Zarządzania, przechodząc do zakładki Support i klikając `Download diagnostics info`.
- [ ] Używaj polecenia `ghe-support-bundle` do stworzenia i przesłania pliku diagnostycznego.
- [ ] Wygeneruj plik diagnostyczny za pomocą aplikacji mobilnej GitHub.
- [ ] Uruchom polecenie `ghe-diagnostics -u`, aby stworzyć i automatycznie przesłać plik diagnostyczny do GitHub Support.
