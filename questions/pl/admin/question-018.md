---
question: "W jaki sposób są tworzone konta użytkowników w przypadku użytkowników zarządzanych przez Enterprise?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/concepts/identity-and-access-management/enterprise-managed-users"
---

- [x] Konta użytkowników są tworzone przez IdP przedsiębiorstwa, a dostęp do GitHub Enterprise Cloud jest przyznawany.
- [ ] Konta użytkowników są tworzone ręcznie przez każdego użytkownika na GitHub.com.
- [ ] Konta użytkowników są tworzone przez GitHub bez żadnego wkładu ze strony IdP przedsiębiorstwa.
- [ ] Konta użytkowników są automatycznie generowane na podstawie publicznych adresów e-mail.
