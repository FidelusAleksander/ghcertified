---
question: "Jak możesz zweryfikować, czy Twój GitHub self-hosted-runner ma dostęp do wszystkich wymaganych usług GitHub?"
documentation: "https://docs.github.com/en/actions/hosting-your-own-runners/managing-self-hosted-runners/monitoring-and-troubleshooting-self-hosted-runners#checking-self-hosted-runner-network-connectivity"
---

- [x] Korzystając ze skryptu udostępnionego przez GitHub na maszynie runnera
- [ ] Poprzez próbę uzyskania dostępu do maszyny runnera za pomocą `ssh`, aby zweryfikować łączność sieciową
- [ ] Korzystając z predefiniowanego GitHub Actions workflow `network-connectivity.yml`
- [ ] GitHub automatycznie zweryfikuje łączność sieciową podczas instalowania aplikacji runnera na maszynie runnera
