---
question: "Jak można zastąpić organizacyjny GitHub Secret `API_KEY` inną wartością podczas pracy w ramach repozytorium?"
documentation: "https://docs.github.com/en/actions/reference/security/secrets"
---

- [x] Tworząc sekretny element repozytorium o tej samej nazwie `API_KEY`
- [x] Tworząc sekretny element środowiska o tej samej nazwie `API_KEY`
- [ ] Tworząc sekretny element przedsiębiorstwa o tej samej nazwie `API_KEY`
- [ ] Tworząc sekretny element przedsiębiorstwa o nazwie `OVERRIDE_API_KEY`
- [ ] Tworząc sekretny element repozytorium o nazwie `OVERRIDE_API_KEY`
- [ ] Tworząc sekretny element środowiska o nazwie `OVERRIDE_API_KEY`
- [ ] Tworząc sekretny element repozytorium o nazwie `REPOSITORY_API_KEY`
- [ ] Tworząc sekretny element środowiska o nazwie `ENVIRONMENT_API_KEY`
