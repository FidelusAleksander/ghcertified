question: "Które z poniższych stwierdzeń dotyczących przypiętych elementów na Twoim profilu GitHub jest prawdziwe?"  
draft: false  
documentation: "https://docs.github.com/en/account-and-profile/how-tos/profile-customization/pinning-items-to-your-profile"

- [ ] Możesz przypiąć do 10 repozytoriów na swoim profilu.  
- [x] Możesz przypiąć do sześciu repozytoriów lub gistów na swoim profilu, aby mieć do nich szybki dostęp.  
> **Poprawne**. GitHub pozwala na przypięcie do sześciu elementów (repozytoriów lub gistów).  
- [ ] Możesz przypinać tylko publiczne repozytoria, a nie prywatne.  
- [ ] Przypięcie elementu usuwa go z listy repozytoriów.
