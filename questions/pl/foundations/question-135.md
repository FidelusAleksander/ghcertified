- [ ] GitHub Pages pozwala na hostowanie prywatnej strony internetowej za darmo, dostępnej tylko dla członków organizacji.  
> Strony GitHub Pages są zawsze publicznie dostępne, chyba że używasz strony projektowej z regułami zapory ogniowej w GitHub Enterprise Server.  
- [x] GitHub Pages umożliwia hostowanie witryn statycznych z repozytorium GitHub.  
> **Poprawne**. GitHub Pages służy konkretne do hostowania witryn statycznych z Twojego repozytorium.  
- [ ] GitHub Pages automatycznie konwertuje pliki Java na dynamiczne aplikacje webowe.  
- [ ] GitHub Pages mogą być używane wyłącznie przez konta organizacji.  
> Zarówno konta osobiste, jak i organizacji mogą korzystać z Pages.
