- [ ] GitHub Pages pozwala na hostowanie prywatnej strony internetowej za darmo, dostępnej tylko dla członków organizacji.  
> Strony GitHub Pages są zawsze publicznie dostępne, chyba że używasz witryny projektowej z regułami zapory sieciowej dla przedsiębiorstw na GitHub Enterprise Server.  
- [x] GitHub Pages umożliwia hostowanie stron statycznych z repozytorium GitHub.  
> **Poprawne**. GitHub Pages jest przeznaczone specjalnie do hostowania stron statycznych z Twojego repozytorium.  
- [ ] GitHub Pages automatycznie konwertuje pliki Java na dynamiczne aplikacje internetowe.  
- [ ] GitHub Pages może być używane tylko przez konta organizacji.  
> Zarówno konta osobiste, jak i organizacyjne mogą korzystać z Pages.
