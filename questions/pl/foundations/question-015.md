---
question: "Które z poniższych stwierdzeń dotyczących `saved replies` są prawdziwe?"
documentation: "https://docs.github.com/en/get-started/writing-on-github/working-with-saved-replies/using-saved-replies"
---

- [x] Możesz tworzyć, edytować i usuwać je w ustawieniach swojego konta GitHub w sekcji `Saved replies`.
- [x] Saved replies to komentarze, które możesz ponownie wykorzystać w issues i pull requests.
- [ ] Gdy ktoś skomentuje Twój issue lub pull request, możesz zapisać jego odpowiedź i ustawić powiadomienie, aby przypomnieć sobie o jej późniejszym odpisaniu.
- [ ] Saved replies są dostępne tylko dla właścicieli repository, którzy mogą skonfigurować automatyczne odpowiedzi na issues i pull requests.
