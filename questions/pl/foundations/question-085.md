question: "W jaki sposób GitHub pomaga osobom, które chcą pisać właściwe komentarze w zgłoszeniach (issue) i pull requestach, ale nie znają składni Markdown?"
documentation: "https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/about-writing-and-formatting-on-github#text-formatting-toolbar"

- [x] Na formularzach komentarzy w zgłoszeniach (issue) i pull requestach znajduje się pasek formatowania tekstu, który generuje dla ciebie Markdown.
- [ ] Udostępnienie arkusza pomocy dla Markdown.
- [ ] GitHub automatycznie konwertuje HTML na Markdown w komentarzach zgłoszeń (issue) i pull requestów.
- [ ] Zgłoszenia (issue) i pull requesty na GitHubie nie obsługują Markdown.
