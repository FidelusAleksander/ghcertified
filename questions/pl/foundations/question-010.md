---
question: "Jaka jest różnica między GitHub projects a GitHub projects classic?"
documentation: "https://docs.github.com/en/issues/planning-and-tracking-with-projects"
---

- [x] GitHub Projects to nowa wersja GitHub, która oferuje wiele nowych funkcji i ulepszeń w porównaniu z wcześniejszą wersją GitHub Projects classic.
- [ ] GitHub Projects classic to nowa wersja GitHub, która oferuje wiele nowych funkcji i ulepszeń w porównaniu z wcześniejszą wersją GitHub Projects.
- [ ] GitHub Projects classic to prostszy sposób zarządzania projektami w GitHub i jest polecany dla początkujących.
- [ ] GitHub Projects classic to ulepszona wersja, która poza GitHub Projects oferuje gotowe szablony, takie jak mapy drogowe i sprinty.
