---
question: "Czym jest plan GitHub Pro?"
documentation: "https://docs.github.com/en/get-started/learning-about-github/githubs-plans#github-pro"
---

- [x] GitHub Pro to płatny plan dla kont osobistych, oferujący dodatkowe funkcje w porównaniu do bezpłatnego planu.
- [ ] GitHub Pro to płatny plan dla organizacji, oferujący dodatkowe funkcje w porównaniu do bezpłatnego planu dla organizacji.
- [ ] GitHub Pro to ścieżka certyfikacji, na końcu której zostajesz GitHub Pro.
- [ ] GitHub Pro to płatny plan dla przedsiębiorstw, umożliwiający samoobsługowe hostowanie GitHub na własnych serwerach.
