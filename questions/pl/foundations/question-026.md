- [x] To prekonfigurowane środowisko programistyczne specjalnie przygotowane dla repozytorium. Umożliwia natychmiastowe rozpoczęcie pisania kodu dla projektu bez konieczności konfigurowania lokalnego środowiska programistycznego.
- [ ] GitHub Codespace to interaktywne holograficzne środowisko kodowania, które wymaga specjalnego sprzętu do użycia.
- [ ] GitHub Codespace to narzędzie do kodowania zasilane przez AI, które automatycznie generuje niemal bezbłędny kod.
- [ ] GitHub Codespace to coroczne wydarzenie organizowane przez GitHub, podczas którego deweloperzy mogą się spotykać i omawiać tematy związane z programowaniem.
