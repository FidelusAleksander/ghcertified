---
question: "Które z tych stwierdzeń najlepiej opisuje, czym jest GitHub?"
documentation: "https://docs.github.com/en/get-started/using-git/about-git#how-github-works"
---

- [x] GitHub to platforma, która wykorzystuje Git jako swoją podstawową technologię i dodaje funkcje ułatwiające współpracę oraz zarządzanie kodem. 
- [ ] GitHub to rozproszony system kontroli wersji z funkcjami, które ułatwiają współpracę nad projektami.
- [ ] GitHub to platforma do udostępniania plików z funkcjami, które ułatwiają współpracę nad projektami, wykorzystując scentralizowaną kontrolę wersji.
- [ ] GitHub to nowoczesna platforma CI/CD z funkcjami wspierającymi szybki rozwój produktów.
