---
question: "Czym są Enterprise Managed Users?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/identity-and-access-management/understanding-iam-for-enterprises/about-enterprise-managed-users#about-enterprise-managed-users"
---

- [x] Jest to funkcja GitHub Enterprise, która pozwala zarządzać dostępem użytkowników za pomocą zewnętrznych dostawców tożsamości, takich jak Azure AD lub Okta
- [ ] Jest to użytkownik wsparcia GitHub Enterprise, który może być używany do zarządzania Twoim Enterprise przez GitHub Support na żądanie
- [ ] Jest to współdzielone konto użytkownika, które może być używane przez wielu użytkowników w Twoim Enterprise
- [ ] Jest to konto użytkownika bez dostępu do interfejsu webowego GitHub, które jest używane wyłącznie do zadań automatyzacji w Twoim GitHub Enterprise
