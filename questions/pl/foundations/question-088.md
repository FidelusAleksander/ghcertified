---
question: "Jak można powiązać pull request z issue?"
documentation: "https://docs.github.com/en/issues/tracking-your-work-with-issues/linking-a-pull-request-to-an-issue"
---

- [x] Ręcznie z poziomu pull request lub widoku issue.
- [x] Poprzez użycie słowa kluczowego w opisie pull request i odniesienie się do numeru issue.
- [ ] Poprzez użycie słowa kluczowego w opisie pull request i odniesienie się do tytułu issue.
- [ ] Poprzez dołączenie numeru issue w zmianach kodu.
- [ ] Poprzez dołączenie numeru issue w opisie commit.
