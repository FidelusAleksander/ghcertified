question: "Czym jest GitHub Marketplace?"
documentation: "https://docs.github.com/en/apps/github-marketplace/github-marketplace-overview/about-github-marketplace-for-apps"
---

- [x] GitHub Marketplace to miejsce, gdzie programiści mogą znaleźć narzędzia pomagające w tworzeniu oprogramowania bezpośrednio obok swojego kodu.
- [ ] GitHub Marketplace to miejsce, gdzie firmy mogą znaleźć programistów do zatrudnienia.
- [ ] GitHub Marketplace to miejsce, gdzie programiści oferują sprzedaż własności swoich repozytoriów.
- [ ] GitHub Marketplace to miejsce, gdzie można sponsorować programistów open-source na zasadzie subskrypcji.
> To jest GitHub Sponsors.
