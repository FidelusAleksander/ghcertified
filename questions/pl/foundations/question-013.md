---
question: "Jak można przypisać osobę do issue lub pull requesta?"
documentation: "https://docs.github.com/en/issues/tracking-your-work-with-issues/using-issues/assigning-issues-and-pull-requests-to-other-github-users"
---

- [x] Korzystając z pola Assignees w panelu bocznym
- [ ] Dodając etykietę z ich nazwą użytkownika GitHub do tego issue lub pull requesta
- [ ] Wspominając ich w opisie issue lub pull requesta
- [ ] Nie można przypisać osoby do issue lub pull requesta.
