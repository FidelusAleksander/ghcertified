---
question: "Jak rozpocząć korzystanie z GitHub Copilot po aktywowaniu subskrypcji GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [x] Skonfiguruj GitHub Copilot w jednym z obsługiwanych IDE, takich jak Visual Studio Code lub JetBrains, i zacznij kodować
- [ ] GitHub Copilot automatycznie zacznie sugerować zmiany w pull requests i issue w Twoim repozytorium
- [ ] Musisz skonfigurować GitHub Action, który skonfiguruje GitHub Copilot w Twoim repozytorium
- [ ] Musisz edytować ustawienia repozytorium i włączyć GitHub Copilot dla repozytorium
