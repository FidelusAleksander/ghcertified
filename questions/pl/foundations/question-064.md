- [x] Szablon Pull Request to plik definiujący domyślną strukturę Pull Request, którą automatycznie widzą współtwórcy projektu podczas otwierania Pull Request.
- [ ] Szablon Pull Request to Pull Request automatycznie generowany podczas tworzenia nowego branch.
- [ ] Szablon Pull Request to Pull Request wybrany przez właściciela repozytorium jako odniesienie przy tworzeniu nowych Pull Request.
- [ ] Szablon Pull Request to przykład, jak zmiany w kodzie powinny wyglądać w Pull Request w danym repozytorium.
