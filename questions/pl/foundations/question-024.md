---
question: "Jakie są różnice między GitHub Copilot Individual a GitHub Copilot Business?"
documentation: "https://docs.github.com/en/copilot/about-github-copilot/subscription-plans-for-github-copilot"
---

- [x] GitHub Copilot Individual może być używany tylko przez konta osobiste, a GitHub Copilot Business jest używany przez organizacje i przedsiębiorstwa
- [ ] GitHub Copilot Individual wykorzystuje model AI szkolony na publicznym kodzie źródłowym, a GitHub Copilot Business korzysta z modelu AI szkolonego na prywatnym kodzie źródłowym
- [ ] GitHub Copilot Individual jest darmowy, a GitHub Copilot Business to funkcja płatna
- [ ] GitHub Copilot Individual może być używany tylko podczas pracy z publicznymi repozytoriami, a GitHub Copilot Business może być używany podczas pracy z publicznymi i prywatnymi repozytoriami w ramach organizacji
