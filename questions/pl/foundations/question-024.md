---
question: "Jakie są różnice między GitHub Copilot Individual a GitHub Copilot Business?"
documentation: "https://docs.github.com/en/copilot/get-started/plans"
---

- [x] GitHub Copilot Individual może być używany tylko przez konta osobiste, a GitHub Copilot Business jest używany przez organizacje i przedsiębiorstwa
- [ ] GitHub Copilot Individual korzysta z modelu AI trenowanego na publicznym kodzie źródłowym, a GitHub Copilot Business korzysta z modelu AI trenowanego na prywatnym kodzie źródłowym
- [ ] GitHub Copilot Individual jest bezpłatny, a GitHub Copilot Business jest funkcją płatną
- [ ] GitHub Copilot Individual może być używany tylko podczas pracy z publicznymi repozytoriami, a GitHub Copilot Business może być używany podczas pracy z publicznymi i prywatnymi repozytoriami w organizacji
