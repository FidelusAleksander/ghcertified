---
question: "Kto może włączyć alerty Dependabot w repozytorium?"
documentation: "https://docs.github.com/en/code-security/concepts/supply-chain-security/dependabot-alerts"
---

- [x] Właściciele repozytorium i osoby z dostępem administratora
- [ ] Tylko właściciel repozytorium 
- [ ] Alerty Dependabot są włączane na wszystkich repozytoriach przez GitHub i nie mogą być wyłączane ani włączane przez żadną osobę.
- [ ] Alerty Dependabot są włączane przez dodanie GitHub Action do repozytorium, więc każda osoba z dostępem do zapisu w repozytorium może je włączyć.
