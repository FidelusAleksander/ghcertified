---
question: "Kto może włączyć alerty Dependabot w repozytorium?"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/dependabot-alerts/about-dependabot-alerts#configuration-of-dependabot-alerts"
---

- [x] Właściciele repozytoriów i osoby z dostępem administracyjnym
- [ ] Tylko właściciel repozytorium 
- [ ] Alerty Dependabot są włączone we wszystkich repozytoriach przez GitHub i nie mogą być wyłączane ani włączane przez żadną osobę.
- [ ] Alerty Dependabot są włączane przez dodanie GitHub Action do repozytorium, więc każdy z uprawnieniami do zapisu w repozytorium może je włączyć.
