---
question: "Czy wykres zależności GitHub jest dostępny za darmo dla wszystkich repozytoriów?"
documentation: https://docs.github.com/en/code-security/concepts/supply-chain-security/dependency-graph#dependency-graph-availability
---

- [x] Tak, jest dostępny za darmo dla wszystkich repozytoriów.
- [ ] Nie, jest dostępny za darmo tylko dla publicznych repozytoriów. Prywatne repozytoria mogą go używać, jeśli posiadają licencję GitHub Advanced Security.
