---
question: "Przypadkowo wprowadziłeś do publicznego repozytorium swój GitHub personal access token. Jakie działania powinieneś podjąć, aby zapobiec kompromitacji swojego konta?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning#securing-compromised-secrets"
---

- [x] Uznaj token za skompromitowany i natychmiast go usuń
- [ ] Zmień uprawnienia tokena na tylko do odczytu
- [ ] Nadpisz historię git, aby ukryć token
- [ ] Sprawdź, czy token jest używany w którejkolwiek z Twoich aplikacji, jeśli tak - usuń go.
