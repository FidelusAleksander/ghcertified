- [x] Skanowanie kodu może być uruchamiane przez różne zdarzenia zachodzące w repository.
- [ ] Skanowanie kodu jest uruchamiane przy każdym push do repository.
- [ ] Skanowanie kodu jest uruchamiane według konfigurowalnego harmonogramu.
- [ ] Skanowanie kodu może być uruchamiane według konfigurowalnego harmonogramu lub w pull requests.
