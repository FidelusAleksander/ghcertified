- [x] Przejdź do ustawień organizacji `Code security and analysis` i włącz aktualizacje bezpieczeństwa Dependabot dla wszystkich repozytoriów jednocześnie.
- [ ] Ustaw wszystkie repozytoria w organizacji jako prywatne.
- [ ] Uruchom akcję GitHub `actions/enable-ghas` z parametrem `security-updates` ustawionym na `true` dla wszystkich repozytoriów w organizacji.
- [ ] Utwórz skrypt, który włączy aktualizacje bezpieczeństwa Dependabot dla wszystkich repozytoriów w organizacji.
