- [x] Częściowo opierają się na GitHub Advisory Database
- [x] Aby włączyć Dependabot Alerts, najpierw musisz mieć włączony Dependency Graph w swoim repozytorium
- [x] Gdy GitHub wykryje podatną zależność, generuje alert Dependabot i wyświetla go na karcie Security dla repozytorium
- [ ] Dependabot Alerts są domyślnie włączone dla wszystkich repozytoriów
- [ ] Dependabot Alerts są domyślnie włączone dla wszystkich publicznych repozytoriów
- [ ] Dependabot Alerts informują Cię, że Twoje repozytorium używa przestarzałej wersji pakietu
