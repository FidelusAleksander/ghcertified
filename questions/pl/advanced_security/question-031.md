- [x] Częściowo opierają się na GitHub Advisory Database
- [x] Aby włączyć Dependabot Alerts, najpierw musisz mieć włączony Dependency Graph w swoim repository
- [x] Gdy GitHub wykryje podatną zależność, generuje alert Dependabot i wyświetla go na karcie Security w repository
- [ ] Dependabot Alerts są domyślnie włączone dla wszystkich repository
- [ ] Dependabot Alerts są domyślnie włączone dla wszystkich publicznych repository
- [ ] Dependabot alerts informują, że Twoje repository używa nieaktualnej wersji pakietu
