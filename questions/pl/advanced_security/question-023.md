---
question: "Jakie informacje dostarczają alerty Dependabot?"
documentation: "https://docs.github.com/en/code-security/concepts/supply-chain-security/dependabot-alerts"
---

- [x] Alerty Dependabot informują, że Twój repository używa niebezpiecznego pakietu.
- [ ] Alerty Dependabot informują, że Twój repository jest używany przez inne publiczne repository.
- [ ] Alerty Dependabot informują, że Twój repository używa nietestowanej wersji pakietu.
- [ ] Alerty Dependabot informują, że Twój repository używa nieaktualnej wersji pakietu.
> Dependabot zajmuje się tym za pomocą aktualizacji wersji, ale nie za pomocą alertów.
