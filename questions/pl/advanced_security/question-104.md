documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/dependabot-alerts/configuring-dependabot-alerts#enabling-or-disabling-dependabot-alerts-for-all-existing-repositories"

- [x] Przejść do ustawień `Code security and analysis` organizacji i włączyć Dependabot Alerts dla wszystkich repozytoriów jednocześnie.
- [ ] Ustawić wszystkie repozytoria w organizacji jako prywatne.
- [ ] Na wszystkich repozytoriach w organizacji uruchomić GitHub Action `actions/enable-ghas` z parametrem `alerts` ustawionym na `true`. 
- [ ] Utworzyć skrypt, który włączy Dependabot Alerts we wszystkich repozytoriach organizacji.
