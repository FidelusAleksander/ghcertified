- [x] Przejść do ustawień `Code security and analysis` organizacji i włączyć Dependabot Alerts dla wszystkich repozytoriów jednocześnie.
- [ ] Zmienić wszystkie repozytoria w organizacji na prywatne.
- [ ] W każdym repozytorium w organizacji uruchomić GitHub Action `actions/enable-ghas` z parametrem `alerts` ustawionym na `true`.
- [ ] Utworzyć skrypt, który włączy Dependabot Alerts w wszystkich repozytoriach organizacji.
