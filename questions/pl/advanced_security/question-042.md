- [ ] Skanowanie sekretów przeszukuje całą historię Git na wszystkich gałęziach obecnych w Twoim repozytorium GitHub w poszukiwaniu sekretów.
- [ ] Skanowanie sekretów przeszukuje tytuły, opisy i komentarze w otwartych i zamkniętych historycznych zgłoszeniach w poszukiwaniu sekretów.
- [ ] Skanowanie sekretów może zapobiegać przesyłaniu obsługiwanych sekretów do Twojego Enterprise, Organization lub Repository.
