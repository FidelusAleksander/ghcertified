- [x] Skanowanie sekretów to narzędzie do bezpiecznego przechowywania i zarządzania sekretami.
- [ ] Skanowanie sekretów przeszuka całą historię Git na wszystkich gałęziach obecnych w twoim repozytorium GitHub w poszukiwaniu sekretów.
- [ ] Skanowanie sekretów przeszuka tytuły, opisy i komentarze w otwartych i zamkniętych historycznych zgłoszeniach w poszukiwaniu sekretów.
- [ ] Skanowanie sekretów może zapobiec wprowadzaniu obsługiwanych sekretów do twojego enterprise, organizacji lub repozytorium.
