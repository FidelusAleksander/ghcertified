- [x] W zakładce Security konsoli zarządzania Site admin
- [x] Poprzez bezpośrednie połączenie SSH z instancją GitHub Enterprise Server i użycie poleceń powłoki administracyjnej `ghe-config`.
- [ ] Poprzez zgłoszenie żądania aktualizacji do GitHub Support
- [ ] Ustawiając opcję konfiguracji `github.advanced_security.enabled` na `true` w pliku `config.yml` w katalogu `/etc/github` na instancji GitHub Enterprise Server.
- [ ] Ustawiając opcję konfiguracji `github.advanced_security.enabled` na `true` w pliku `config.yml` w repozytorium `.github`.
