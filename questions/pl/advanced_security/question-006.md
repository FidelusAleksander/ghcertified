question: "Które z tych funkcji bezpieczeństwa GitHub są dostępne ZA DARMO zarówno dla publicznych, jak i prywatnych osobistych repository?"
documentation: "https://docs.github.com/en/code-security/getting-started/github-security-features"
---

- [x] Polityka bezpieczeństwa
- [x] Doradztwo w sprawach bezpieczeństwa
- [x] Powiadomienia Dependabot i aktualizacje dotyczące bezpieczeństwa
- [x] Aktualizacje wersji Dependabot
- [ ] Skanowanie kodu Dependabot
> Dependabot nie jest narzędziem do skanowania kodu, lecz narzędziem do zarządzania zależnościami
- [ ] Skanowanie tajnych danych Dependabot
> Dependabot nie jest narzędziem do skanowania tajnych danych, lecz narzędziem do zarządzania zależnościami
- [ ] Skanowanie tajnych danych
> Skanowanie tajnych danych jest dostępne dla publicznych repository oraz dla organizacji posiadających licencję GitHub Advanced Security. Nie jest dostępne dla osobistych prywatnych repository.
- [ ] Skanowanie kodu
> Skanowanie kodu jest dostępne dla publicznych repository oraz dla organizacji posiadających licencję GitHub Advanced Security. Nie jest dostępne dla osobistych prywatnych repository.
