---
question: "Jak można używać CodeQL w zewnętrznym systemie CI razem z repozytoriami GitHub?"
documentation: https://docs.github.com/en/code-security/how-tos/find-and-fix-code-vulnerabilities/integrate-with-existing-tools/use-with-existing-ci-system#about-using-code-scanning-with-your-existing-ci-system
---

- [x] Uruchom CodeQL CLI w zewnętrznym systemie CI, aby skanować kod i przesyłać wyniki do repozytorium GitHub
- [ ] CodeQL nie może być używane w zewnętrznych systemach CI; jest dostępne wyłącznie dla GitHub Actions
- [ ] Prześlij kod źródłowy do GitHub w celu analizy, a następnie pobierz wyniki do użycia w systemie CI
- [ ] Ręcznie uruchom CodeQL lokalnie i wyślij wyniki e-mailem do administratorów repozytorium GitHub
