---
question: "Jak można ustawić domyślną politykę bezpieczeństwa dla wszystkich repozytoriów w Organizacji GitHub `my-org`?"
documentation: "https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file#supported-file-types"
---

- [x] Poprzez utworzenie pliku `SECURITY.md` w repozytorium `my-org/.github`
- [ ] Poprzez edycję polityki bezpieczeństwa w ustawieniach organizacji `Code Security and analysis`
- [ ] Domyślne polityki bezpieczeństwa mogą być ustawiane tylko przez wsparcie GitHub
- [ ] Można ustawić domyślną politykę bezpieczeństwa dla wszystkich repozytoriów w Organizacji GitHub `my-org`, dodając plik `SECURITY.md` do każdego indywidualnego repozytorium.
