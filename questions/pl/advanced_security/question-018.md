---
question: "Kiedy GitHub przeprowadza skanowanie sekwencji wszystkich historycznych kodów w repozytoriach enterprise, jakie jest zachowanie powiadomień?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning/monitoring-alerts#historical-scans"
---

- [x] GitHub powiadamia właścicieli enterprise i menedżerów bezpieczeństwa, nawet jeśli nie znaleziono żadnych sekretów.
- [x] GitHub powiadamia administratorów Repository, menedżerów bezpieczeństwa oraz użytkowników z niestandardowymi rolami z dostępem do odczytu/zapisu, za każdym razem, gdy sekret zostanie wykryty w repozytorium.
- [ ] GitHub powiadamia właścicieli enterprise i menedżerów bezpieczeństwa tylko wtedy, gdy wykryje ujawnione sekrety.
- [ ] GitHub powiadamia autorów commitów, które zawierają ujawnione sekrety.
