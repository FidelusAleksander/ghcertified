- [x] Usando GitHub Connect
- [ ] GitHub Enterprise Server tiene acceso a todas las acciones de GitHub.com por defecto
> GitHub Actions en GitHub Enterprise Server está diseñado para funcionar en entornos sin acceso completo a internet. Por defecto, los workflows no pueden usar acciones de GitHub.com ni de GitHub Marketplace.
- [ ] Usando la herramienta actions-sync
