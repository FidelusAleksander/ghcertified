---
question: "¿Qué es verdadero sobre `Starter Workflows`?"
documentation: "https://docs.github.com/en/actions/how-tos/reuse-automations/create-workflow-templates"
---

- [x] Permiten a los usuarios utilizar plantillas de flujo de trabajo listas para usar (o que requieren cambios mínimos)
- [x] GitHub proporciona y mantiene los starter workflows para diferentes categorías, lenguajes y herramientas
- [x] Tu organización puede crear starter workflows personalizados para los usuarios de tu organización
- [ ] Los starter workflows no pueden llamar a flujos de trabajo reutilizables
- [ ] Los starter workflows son una función paga de GitHub
- [ ] Los starter workflows se proporcionan listos para usar y no pueden modificarse ni mejorarse
> https://docs.github.com/en/actions/using-workflows/using-starter-workflows#using-starter-workflows
