---
question: "¿Qué componentes se pueden reutilizar dentro de una GitHub Organization?"
---

- [x] Secrets
- [x] Configuration Variables
- [x] Self Hosted Runners
- [x] Workflow Templates
- [ ] Artifacts
> Los Artifacts se utilizan para preservar datos después de que un trabajo ha finalizado y/o compartir esos datos con otro trabajo dentro del mismo workflow. 
- [ ] Cache
> El Cache puede ser reutilizado en workflows dentro de un solo repository.
- [ ] Environment Variables
> Las Environment Variables pueden ser configuradas en el nivel de un paso, trabajo o workflow. No pueden ser compartidas entre workflows, repositories u organizations.
