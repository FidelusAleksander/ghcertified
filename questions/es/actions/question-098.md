---
question: "¿Cómo puedes especificar el horario de un flujo de trabajo de GitHub Actions para que se ejecute solo en días de semana?"
documentation: "https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#schedule"
---

- [ ] agregar una condición en el archivo YAML del flujo de trabajo para los días de semana
- [ ] no es posible en GitHub Actions
- [ ] usar el disparador de evento on: schedule: weekdays
- [x] usar el disparador de evento on: schedule: cron
