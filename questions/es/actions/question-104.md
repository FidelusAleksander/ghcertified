---
question: "¿Cuál es la práctica recomendada para tratar las variables de entorno en GitHub Actions, independientemente del sistema operativo y shell utilizado?"
documentation: "https://docs.github.com/en/actions/writing-workflows/choosing-what-your-workflow-does/workflow-commands-for-github-actions#setting-an-environment-variable"
---

- [x] tratar las variables de entorno como sensibles a mayúsculas y minúsculas
- [ ] usar solo letras mayúsculas para los nombres de las variables de entorno
- [ ] ignorar la sensibilidad a mayúsculas y minúsculas ya que GitHub Actions lo maneja automáticamente
- [ ] depender del comportamiento del sistema operativo en uso
