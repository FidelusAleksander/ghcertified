---
question: "Los secretos y las variables de configuración pueden aplicarse a:"
---

- [x] Toda la organización o repositorios seleccionados en una organización
- [x] Un único repositorio
- [x] Un entorno en un repositorio
- [ ] Un entorno compartido entre múltiples repositorios
> Los entornos no se pueden compartir entre repositorios
- [ ] Múltiples repositorios que no comparten una organización/empresa
- [ ] Un workflow específico en un repositorio
> Las variables de entorno pueden aplicarse a un workflow, las variables de configuración no
- [ ] Un trabajo específico en un workflow
> Las variables de entorno pueden aplicarse a un workflow, las variables de configuración no
