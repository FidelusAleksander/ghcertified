question: "Los secretos y las variables de configuración pueden limitarse a:"
documentation: "https://docs.github.com/en/actions/using-workflows/sharing-workflows-secrets-and-runners-with-your-organization#sharing-secrets-and-variables-within-an-organization"

- [x] Toda la organización, o repositorios seleccionados dentro de una organización
- [x] Un único repositorio
- [x] Un entorno dentro de un repositorio
- [ ] Un entorno compartido entre varios repositorios  
> Los entornos no pueden compartirse entre repositorios
- [ ] Varios repositorios que no comparten una organización/enterprise
- [ ] Un workflow específico dentro de un repositorio  
> Las variables de entorno pueden limitarse a un workflow, las variables de configuración no
- [ ] Un job específico dentro de un workflow  
> Las variables de entorno pueden limitarse a un workflow, las variables de configuración no
