---
question: "¿Qué es cierto acerca de las exclusiones de contenido de Copilot?"
documentation: "https://docs.github.com/en/copilot/how-tos/configure-content-exclusion/exclude-content-from-copilot"
---

- [x] Las exclusiones de contexto se pueden configurar a nivel de repository y organization
- [x] Copilot ofrece diferentes planes con consideraciones de privacidad
- [ ] Copilot ignora completamente los archivos excluidos
> Copilot puede usar información de un archivo excluido si la información es proporcionada por el IDE.
- [ ] Las exclusiones de contenido no afectan la autocompletación del código
- [ ] Las exclusiones de contenido se aplican instantáneamente
> Después de agregar o cambiar las exclusiones de contenido, puede tardar hasta 30 minutos en surtir efecto
