---
question: "¿Cuál de las siguientes opciones NO es una forma posible de otorgar acceso a Copilot para los miembros de una organización?"
documentation: "https://docs.github.com/en/copilot/managing-copilot/managing-github-copilot-in-your-organization/managing-access-to-github-copilot-in-your-organization/granting-access-to-copilot-for-members-of-your-organization"
---

- [x] Como miembro de una organización, puedes habilitar Copilot directamente desde la configuración de tu cuenta.
- [ ] A través de la configuración de tu Enterprise, habilita GitHub Copilot para organizaciones seleccionadas o todas las organizaciones.
- [ ] A través de la configuración de tu organización, habilita GitHub Copilot para equipos seleccionados o usuarios específicos, o para toda la organización.
- [ ] Puedes utilizar la REST API de GitHub para otorgar acceso a GitHub Copilot a equipos o usuarios específicos en tu organización.
