---
question: "¿Qué opción a continuación NO es una forma posible de otorgar acceso a Copilot para miembros de una organización?"
documentation: "https://docs.github.com/en/copilot/how-tos/administer-copilot/manage-for-organization/manage-access/grant-access"
---

- [x] Como miembro de una organización, puedes habilitar Copilot directamente desde la configuración de tu cuenta.
- [ ] A través de la configuración de tu Enterprise, habilita GitHub Copilot para organizaciones seleccionadas o para todas las organizaciones.
- [ ] A través de la configuración de tus organizaciones, habilita GitHub Copilot para equipos seleccionados o usuarios específicos o para toda la organización.
- [ ] Puedes usar la REST API de GitHub para otorgar acceso a GitHub Copilot a equipos o usuarios específicos en tu organización.
