question: "¿Qué afirmación describe correctamente GitHub Copilot CLI?"
documentation: "https://docs.github.com/en/copilot/concepts/agents/copilot-cli/about-copilot-cli"

- [x] Te permite usar Copilot desde tu terminal para responder preguntas, escribir y depurar código, e interactuar con GitHub.com
- [ ] Está limitado a generar alias de shell para `gh copilot suggest` y `gh copilot explain`
- [ ] Solo se ejecuta dentro de la interfaz web de GitHub y no puede acceder a archivos de proyectos locales
- [ ] Ejecuta automáticamente cada comando de shell sugerido sin solicitar aprobación
