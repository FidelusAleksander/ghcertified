---
question: "¿Qué afirmación describe correctamente la funcionalidad del comando CLI de GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/using-github-copilot/using-github-copilot-in-the-command-line"
---

- [x] Los usuarios pueden obtener explicaciones de comandos usando 'gh copilot explain' y sugerencias de comandos usando 'gh copilot suggest'
- [ ] Los comandos se ejecutan automáticamente sin confirmación del usuario al usar 'gh copilot suggest'
- [ ] El comando 'gh copilot explain' modifica archivos del sistema sin mostrar la explicación
- [ ] Los comandos sugeridos se ejecutan directamente sin ser copiados al portapapeles primero
