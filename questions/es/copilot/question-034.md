- [x] Las indicaciones no se retienen, mientras que los análisis de uso se retienen de forma predeterminada.
- [x] Es posible optar por no participar en los análisis de uso opcionales.
- [x] Las respuestas generadas por Copilot CLI pueden revisarse después de cada sugerencia.
- [ ] Tanto las indicaciones como los análisis de uso se retienen de forma predeterminada.
> Las indicaciones no se retienen de forma predeterminada.
- [ ] No es posible optar por no participar en los análisis de uso opcionales porque son anónimos.
- [ ] GitHub Copilot CLI solo puede explicar o sugerir comandos, pero no ejecutarlos en nombre del usuario.
