---
question: "¿Cómo gestiona GitHub Copilot la retención de datos para las sugerencias de código en el IDE?"
documentation: "https://learn.github.com/learning-pathways/github-copilot"
---

- [x] Las sugerencias se mantienen temporalmente en memoria y se descartan después de usarlas, no se escriben en el disco
- [ ] Todas las sugerencias se almacenan permanentemente en una base de datos local para referencia futura
- [ ] Las sugerencias se guardan automáticamente en repositorios de GitHub para control de versiones
- [ ] Los fragmentos de código se almacenan en caché en el disco durante 30 días antes de ser eliminados
