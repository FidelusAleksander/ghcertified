---
question: "¿Para qué se utiliza un alias al usar GitHub Copilot?"
documentation: "https://learn.microsoft.com/en-us/training/modules/github-copilot-across-environments/4-git-hub-copilot-for-the-command-line"
---

- [x] Usar un alias permite que GitHub Copilot CLI ejecute automáticamente comandos en la línea de comandos.
- [ ] Usar un alias te permite indicar en los commits de Git que GitHub Copilot realizó cambios.
- [ ] Usar un alias permite que GitHub revise y cree pull requests en tu nombre.
