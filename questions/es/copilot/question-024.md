---
question: "¿Cómo funciona la característica de coincidencia de código público de GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/using-github-copilot/finding-public-code-that-matches-github-copilot-suggestions"
---

- [x] Busca coincidencias comparando las sugerencias de código con un índice de repositorios públicos de GitHub, que se actualiza cada pocos meses
- [ ] Realiza búsquedas en tiempo real en todos los repositorios de GitHub, incluyendo los privados
- [ ] Solo coincide con código de repositorios que se crearon en las últimas 24 horas
- [ ] Verifica código en plataformas externas de alojamiento de código fuera de GitHub
