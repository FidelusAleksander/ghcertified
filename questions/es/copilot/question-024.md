---
question: "¿Cómo funciona la función de coincidencia de código público de GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/how-tos/get-code-suggestions/find-matching-code"
---

- [x] Busca coincidencias comparando sugerencias de código con un índice de repositorios públicos de GitHub, que se actualiza cada pocos meses
- [ ] Realiza búsquedas en tiempo real en todos los repositorios de GitHub, incluidos los privados
- [ ] Solo coincide con el código de los repositorios que se crearon en las últimas 24 horas
- [ ] Verifica el código en plataformas de alojamiento de código externas a GitHub
