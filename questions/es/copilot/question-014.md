---
question: "¿Cuál es la forma más fácil de comenzar con GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [ ] Solicitar acceso al soporte de GitHub y esperar la aprobación antes de usar GitHub Copilot.
- [ ] Usar el sitio web de Copilot y pegar tu código al pedir sugerencias.
- [x] Instalar la extensión de Copilot en tu entorno preferido, como Visual Studio Code.
- [ ] Crear un nuevo repositorio público de GitHub y habilitar Copilot para que escanee tu código y haga sugerencias.
