---
question: "¿Qué permisos tiene un administrador de GitHub App dentro de una organización?"
documentation: "https://docs.github.com/en/organizations/managing-programmatic-access-to-your-organization/adding-and-removing-github-app-managers-in-your-organization#about-github-app-managers"
---

- [x] Los administradores de GitHub App pueden gestionar la configuración de los registros de GitHub App que pertenecen a la organización, pero no tienen permisos para instalar o desinstalar GitHub Apps.
- [ ] Pueden acceder y modificar el código fuente de cualquier GitHub App propiedad de la organización.
- [ ] Los administradores de GitHub App son los únicos roles que pueden crear nuevas GitHub Apps dentro de la organización.
- [ ] Tienen la capacidad de otorgar permisos de instalación de aplicaciones a colaboradores externos.
