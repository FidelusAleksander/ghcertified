---
question: "¿Qué se requiere para descargar o publicar un paquete de GitHub dentro de workflows, como con GitHub Actions u otras herramientas de CI/CD?"
documentation: "https://docs.github.com/en/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages"
---

- [x] Utilizar `GITHUB_TOKEN` para publicar paquetes asociados con el repositorio del workflow, y un PAT para instalar paquetes desde repositorios privados.
- [ ] GitHub Packages no se pueden usar en workflows de GitHub Actions ni en ninguna herramienta de CI/CD.
- [ ] Solo se requieren claves SSH para usar GitHub Packages en workflows.
- [ ] Se requiere la descarga y carga manual de paquetes para cada ejecución del workflow.
