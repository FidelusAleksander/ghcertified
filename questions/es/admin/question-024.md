---
question: "¿Cuáles son los métodos de autenticación válidos disponibles en GitHub?"
documentation: "https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/about-authentication-to-github"
---

- [x] Nombre de usuario y contraseña (con 2FA opcional)
- [x] Personal access token (PAT)
- [x] Clave SSH
- [x] SAML SSO para cuentas empresariales
- [x] Autenticación Passkey
- [x] Tokens OAuth para integraciones con aplicaciones de terceros
- [ ] Sistema de código QR vinculado a un dispositivo móvil
- [ ] Autenticación a través de cuentas de redes sociales como Facebook o Google
