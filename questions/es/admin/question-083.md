---
question: "Cuando se aplica una lista de IP permitidas de Enterprise, ¿qué ocurre cuando intentas publicar tu sitio de GitHub Pages desde una rama en lugar de utilizar un flujo de trabajo personalizado de GitHub Actions?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/configuration/hardening-security-for-your-enterprise/restricting-network-traffic-to-your-enterprise-with-an-ip-allow-list#using-github-pages-with-an-ip-allow-list"
---

- [x] El ejecutor de la compilación tendrá acceso al repositorio para el sitio de GitHub Pages por defecto
- [ ] El ejecutor de la compilación será bloqueado de acceder al repositorio hasta que se agregue manualmente a la lista de permitidos
- [ ] GitHub Pages no puede ser construido ni actualizado hasta que se desactive la lista de IP permitidas
- [ ] Debes activar manualmente las compilaciones para los sitios de GitHub Pages
