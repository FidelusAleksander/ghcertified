---
question: "¿En qué se diferencia GitHub Enterprise Cloud de GitHub Enterprise Server?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/overview/about-github-for-enterprises#about-deployment-options"
---

- [x] GitHub Enterprise Cloud es alojado por GitHub, mientras que GitHub Enterprise Server es alojado en tus propios servidores o un proveedor de nube.
- [ ] GitHub Enterprise Cloud es alojado en tus servidores, mientras que GitHub Enterprise Server es alojado por GitHub.
- [ ] GitHub Enterprise Cloud ofrece solo repositories públicos, mientras que GitHub Enterprise Server ofrece repositories privados.
- [ ] GitHub Enterprise Cloud está diseñado para uso individual, mientras que GitHub Enterprise Server es para organizaciones.
