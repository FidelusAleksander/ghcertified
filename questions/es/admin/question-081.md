- [x] Ejecutores auto hospedados o ejecutores más grandes alojados por GitHub con rangos de direcciones IP estáticas
- [ ] Solo ejecutores estándar alojados por GitHub
- [ ] Cualquier ejecutor sin considerar la dirección IP
- [ ] Solo ejecutores dentro de la red de la organización sin restricciones de IP
