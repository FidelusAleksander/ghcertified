---
question: "¿Cómo puede un propietario de una organización encontrar estadísticas sobre el uso de licencias en su GitHub Enterprise Server?"
documentation: "https://docs.github.com/en/enterprise-server@3.21/billing/how-tos/manage-plan-and-licenses/view-enterprise-usage"
---

- [x] Navegando al panel de control de `Site admin` para ver el uso de licencias en la configuración de la cuenta `Enterprise`.
- [ ] A través de solicitudes directas por correo electrónico al soporte de GitHub.
- [ ] Verificando la página pública del perfil de la organización para estadísticas en tiempo real.
- [ ] Usando la API de GitHub para obtener datos en tiempo real sobre el uso de licencias en GitHub Enterprise Server.
