---
question: "¿Cuál es el rol de un Security Manager dentro de una organización de GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] Un Security Manager tiene permisos para ver alertas de seguridad y gestionar configuraciones de seguridad del código en toda la organización, además de permisos de lectura para todos los repositories.
- [ ] Los Security Managers son responsables de las transacciones financieras y la seguridad de la información de facturación dentro de la organización.
- [ ] Pueden eliminar o transferir la propiedad de cualquier repository dentro de la organización por razones de seguridad.
- [ ] Los Security Managers gestionan exclusivamente la configuración de autenticación de dos factores para todos los miembros de la organización.
