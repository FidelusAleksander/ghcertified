---
question: "¿Cuál es la diferencia entre GitHub Apps y OAuth apps?"
documentation: "https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/differences-between-github-apps-and-oauth-apps"
---

- [x] GitHub Apps ofrecen permisos granulares a repositorios específicos, mientras que OAuth Apps solicitan acceso a datos de usuario en todos los repositorios.
- [ ] GitHub Apps y OAuth Apps no difieren; proporcionan el mismo nivel de permisos.
- [ ] OAuth Apps ofrecen permisos más granulares en comparación con GitHub Apps.
- [ ] GitHub Apps solo pueden acceder a repositorios públicos, mientras que OAuth Apps pueden acceder tanto a repositorios públicos como privados.
