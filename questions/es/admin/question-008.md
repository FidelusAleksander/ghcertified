---
question: "¿Cuáles son los pasos para instalar una aplicación de GitHub desde GitHub Marketplace para una organización?"
documentation: "https://docs.github.com/en/apps/using-github-apps/installing-a-github-app-from-github-marketplace-for-your-organizations"
---

- [x] Navega por GitHub Marketplace, selecciona la aplicación, elige un plan, selecciona la organización y luego revisa e instala la aplicación.
- [ ] Ve a la configuración de la organización, selecciona "Aplicaciones instaladas" y agrega manualmente el ID de cliente de la aplicación para instalarla.
- [ ] Instala la aplicación utilizando GitHub CLI con el comando `gh app install`, especificando el nombre de la organización.
- [ ] Clona el repositorio de la aplicación y ejecuta el script de instalación localmente para registrarlo con tu organización.
