---
question: "¿Qué pasos deben seguir los administradores de una organización para acceder a los registros de auditoría de las acciones realizadas dentro de su organización en GitHub?"
documentation: "https://docs.github.com/en/organizations/keeping-your-organization-secure/managing-security-settings-for-your-organization/reviewing-the-audit-log-for-your-organization"
---

- [x] Ir a la configuración de la organización > Archivo > Registros > Registro de auditoría. Luego, usar el menú desplegable Exportar para exportar como JSON o CSV.
- [ ] Enviar un correo electrónico directamente al soporte de GitHub para solicitar que se envíe un registro de auditoría mensualmente al correo electrónico del administrador de la organización.
- [ ] Navegar a la página principal de GitHub, usar la barra de búsqueda para escribir "Registros de auditoría de la organización" y esperar que GitHub descargue automáticamente el registro.
- [ ] Revisar los perfiles individuales de los miembros para obtener un resumen de sus acciones, compilar estos resúmenes manualmente y formatearlos para auditoría.
