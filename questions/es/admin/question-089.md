---
question: "¿Qué pasos se deben seguir para crear un grupo de ejecutores autohospedados para una organización en GitHub?"
documentation: "https://docs.github.com/en/actions/how-tos/manage-runners/self-hosted-runners/manage-access"
---

- [x] Navegar a la configuración de la organización, seleccionar Acciones, hacer clic en Grupos de ejecutores, crear un nuevo grupo y asignar una política de acceso al repositorio.
- [ ] En el repositorio `.github`, modificar el archivo `.gitconfig` para especificar grupos de ejecutores y políticas de acceso asociadas.
- [ ] Contactar al soporte de GitHub para solicitar la creación de un grupo de ejecutores y proporcionar una lista de repositorios para la configuración de acceso.
- [ ] Usar un flujo de trabajo de GitHub Actions para generar automáticamente grupos de ejecutores basados en la actividad y patrones de uso del repositorio.
