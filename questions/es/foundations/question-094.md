---
question: "¿Qué son las draft pull requests?"
documentation: "https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests#draft-pull-requests"
---

- [x] Las draft pull requests son pull requests que aún no están listas para ser revisadas.
- [ ] Las draft pull requests son pull requests que ya han sido revisadas pero no fusionadas.
- [ ] Las draft pull requests son pull requests que ya han sido revisadas pero no aprobadas.
- [ ] Las draft pull requests son plantillas para crear nuevas pull requests.
