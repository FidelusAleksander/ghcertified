question: "¿Cuáles son los casos de uso para las etiquetas?"
documentation: "https://docs.github.com/en/issues/using-labels-and-milestones-to-track-work/managing-labels"
---

- [x] Categorizar issues y pull requests
- [ ] Categorizar archivos en el repository
- [ ] Asignarlos a releases para que se incluyan en las notas de la release
> No se pueden asignar etiquetas a las releases
- [ ] Asignar etiquetas a los colaboradores del repository para indicar su papel y permisos en el proyecto
> No se pueden asignar etiquetas a los colaboradores
