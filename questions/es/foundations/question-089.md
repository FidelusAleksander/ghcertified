- [x] Los formularios de problemas permiten que los usuarios sean solicitados a proporcionar información al crear un problema, y luego esa información se agrega automáticamente al problema.
- [ ] No existe algo llamado formularios de problemas en GitHub.
- [ ] El formulario de problemas es una función que cierra automáticamente los problemas después de un cierto período de tiempo, proporcionando una gestión más eficiente de problemas en comparación con las plantillas de problemas.
- [ ] El formulario de problemas es una herramienta para crear problemas que solo contiene campos de formulario para el título y la descripción, simplificando el proceso pero no necesariamente mejorando la calidad de la información en comparación con las plantillas de problemas.
