---
question: "¿Qué es un GitHub Codespace?"
documentation: "https://docs.github.com/en/codespaces/overview"
---

- [x] Es un entorno de desarrollo preconfigurado específicamente configurado para un repositorio. Esto te permite comenzar inmediatamente a escribir código para un proyecto sin tener que configurar un entorno de desarrollo local.
- [ ] Un GitHub Codespace es un entorno de codificación holográfico interactivo que requiere hardware especial para usarlo.
- [ ] GitHub Codespace es una herramienta de codificación impulsada por IA que genera código casi impecable de forma automática.
- [ ] GitHub Codespace es un evento anual organizado por GitHub donde los desarrolladores pueden reunirse y discutir temas de codificación.
