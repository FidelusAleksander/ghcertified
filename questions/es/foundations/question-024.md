---
question: "¿Cuáles son las diferencias entre GitHub Copilot Individual y GitHub Copilot Business?"
documentation: "https://docs.github.com/en/copilot/about-github-copilot/subscription-plans-for-github-copilot"
---

- [x] GitHub Copilot Individual solo puede ser utilizado por cuentas personales y GitHub Copilot Business es usado por organizaciones y empresas
- [ ] GitHub Copilot Individual utiliza un modelo de IA entrenado en código fuente público y GitHub Copilot Business utiliza un modelo de IA entrenado en código fuente privado
- [ ] GitHub Copilot Individual es gratuito y GitHub Copilot Business es una función de pago
- [ ] GitHub Copilot Individual solo se puede utilizar al trabajar en repositorios públicos y GitHub Copilot Business puede ser utilizado al trabajar en repositorios públicos y privados dentro de una organización
