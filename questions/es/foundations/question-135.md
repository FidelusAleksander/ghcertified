- [ ] GitHub Pages te permite alojar un sitio web privado de forma gratuita, accesible solo para los miembros de la organización.  
> Los sitios de GitHub Pages son siempre accesibles públicamente, a menos que uses un sitio de proyecto con reglas de firewall empresarial en GitHub Enterprise Server.  
- [x] GitHub Pages te permite alojar sitios estáticos desde un repositorio de GitHub.  
> **Correcto**. GitHub Pages es específicamente para alojar sitios estáticos desde tu repositorio.  
- [ ] GitHub Pages convierte automáticamente archivos Java en aplicaciones web dinámicas.  
- [ ] GitHub Pages solo puede ser utilizado por cuentas de organización.  
> Tanto las cuentas personales como las cuentas de organización pueden usar Pages.
