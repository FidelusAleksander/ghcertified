- [ ] GitHub Pages te permite alojar un sitio web privado de forma gratuita, accesible solo por miembros de la organización.  
> Los sitios de GitHub Pages siempre son accesibles públicamente, a menos que utilices un sitio de proyecto con reglas de firewall empresarial en GitHub Enterprise Server.  
- [x] GitHub Pages te permite alojar sitios estáticos desde un repositorio de GitHub.  
> **Correcto**. GitHub Pages es específicamente para el alojamiento de sitios estáticos desde tu repositorio.  
- [ ] GitHub Pages convierte automáticamente archivos Java en aplicaciones web dinámicas.  
- [ ] GitHub Pages solo puede ser utilizado por cuentas de organización.  
> Tanto las cuentas personales como las cuentas de organización pueden usar Pages.
