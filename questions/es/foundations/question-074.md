---
question: "¿Qué es un plan GitHub Pro?"
documentation: "https://docs.github.com/en/get-started/learning-about-github/githubs-plans#github-pro"
---

- [x] GitHub Pro es un plan de pago para cuentas personales que ofrece funciones adicionales además del plan gratuito.
- [ ] GitHub Pro es un plan de pago para organizaciones que ofrece funciones adicionales además del plan gratuito para organizaciones.
- [ ] GitHub Pro es una ruta de certificación al final de la cual te conviertes en un GitHub Pro.
- [ ] GitHub Pro es un plan de pago para empresas que permite alojar GitHub en sus propios servidores.
