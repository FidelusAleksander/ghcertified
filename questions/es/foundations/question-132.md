- [ ] Puedes fijar hasta 10 repositorios en tu perfil.  
- [x] Puedes fijar hasta seis repositorios o gists en tu perfil para un acceso rápido.  
> **Correcto**. GitHub admite fijar hasta seis elementos (repositorios o gists).  
- [ ] Solo puedes fijar repositorios públicos, no privados.  
- [ ] Fijar un elemento lo elimina de tu lista de repositorios.
