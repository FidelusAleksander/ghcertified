---
question: "¿Qué son los comandos slash de GitHub?"
documentation: "https://docs.github.com/en/issues/tracking-your-work-with-issues/using-issues/about-slash-commands"
---

- [x] Es una forma de insertar rápidamente Markdown complejo en tus comentarios y descripciones de pull request o issues.
- [ ] Es una forma de automatizar GitHub Actions.
- [ ] Es otro nombre para la GitHub CLI.
- [ ] Es una forma de corregir rápidamente problemas de formato de código en los cambios de código de tu pull request.
