---
question: "¿Cuál de estas afirmaciones sobre las diferencias entre issues y discussions es verdadera?"
documentation: "https://docs.github.com/en/discussions/quickstart"
---

- [x] GitHub Discussions son para conversaciones que necesitan ser transparentes y accesibles, pero que no tienen que ser rastreadas en un tablero del proyecto y no están relacionadas con el código, a diferencia de GitHub Issues.
- [ ] GitHub Issues son para conversaciones que necesitan ser transparentes y accesibles, pero que no tienen que ser rastreadas en un tablero del proyecto y no están relacionadas con el código, a diferencia de GitHub Discussions.
