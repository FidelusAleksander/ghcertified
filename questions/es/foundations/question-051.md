---
question: "¿Qué son los equipos de GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] Es un grupo de usuarios de GitHub de la misma organización.
- [ ] Es una versión de Microsoft Teams para GitHub.
- [ ] Es una forma de gestionar el acceso a los repositorios personales de GitHub para muchos usuarios a la vez.
> Los repositorios personales no se pueden compartir con equipos, solo con colaboradores individuales.
