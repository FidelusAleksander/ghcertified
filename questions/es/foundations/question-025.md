---
question: "¿Cómo puedes empezar a usar GitHub Copilot después de activar la suscripción a GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [x] Configura GitHub Copilot en uno de los IDE compatibles, como Visual Studio Code o JetBrains, y comienza a programar
- [ ] GitHub Copilot comenzará automáticamente a dar sugerencias en los pull requests y issues de tu repositorio
- [ ] Necesitas configurar un GitHub Action que configure GitHub Copilot en tu repositorio
- [ ] Necesitas editar la configuración del repositorio y habilitar GitHub Copilot para el repositorio
