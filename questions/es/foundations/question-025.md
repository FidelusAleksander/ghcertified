---
question: "¿Cómo puedes comenzar a usar GitHub Copilot después de activar la suscripción a GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/using-github-copilot/getting-started-with-github-copilot"
---

- [x] Configura GitHub Copilot en uno de los IDE compatibles, como Visual Studio Code o JetBrains, y comienza a codificar
- [ ] GitHub Copilot comenzará automáticamente a dar sugerencias en los pull requests y issues de tu repositorio
- [ ] Necesitas configurar una GitHub Action que permita configurar GitHub Copilot en tu repositorio
- [ ] Necesitas editar la configuración del repositorio y habilitar GitHub Copilot para el repositorio
