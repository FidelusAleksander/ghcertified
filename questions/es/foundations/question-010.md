---
question: "¿Cuál es la diferencia entre GitHub projects y GitHub projects classic?"
documentation: "https://docs.github.com/en/issues/planning-and-tracking-with-projects"
---

- [x] GitHub Projects es la nueva experiencia de GitHub que ofrece muchas características nuevas y mejoras sobre el antiguo GitHub Projects classic. 
- [ ] GitHub Projects classic es la nueva experiencia de GitHub que ofrece muchas características nuevas y mejoras sobre el antiguo GitHub Projects.
- [ ] GitHub Projects classic es una forma más simple de gestionar tus GitHub Projects y se recomienda para principiantes.
- [ ] GitHub Projects classic es la versión mejorada que, además de GitHub Projects, ofrece algunas plantillas predefinidas como hojas de ruta y sprints.
