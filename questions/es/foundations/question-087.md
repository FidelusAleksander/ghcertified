question: "¿Qué es GitHub Mobile?"
documentation: "https://docs.github.com/en/get-started/using-github/github-mobile"
---

- [x] Es una aplicación móvil para iOS y Android que te permite trabajar con GitHub desde tu teléfono.
- [ ] Es un servicio que te permite probar tus aplicaciones móviles en dispositivos reales.
- [ ] Es un servicio de transferencia de datos de GitHub para petabytes de datos donde un coche propiedad de GitHub irá a tu ubicación y transferirá los datos a tu red local.
- [ ] Es un servicio de telemetría de GitHub que te permite rastrear la actividad de tus usuarios en tu sitio web o aplicación móvil.
