---
question: "Cuando utilizas GitHub Actions como tu sistema de CI y una herramienta de terceros para ejecutar el análisis de código, ¿cómo puedes cargar los resultados SARIF en GitHub?"
documentation: "https://docs.github.com/en/code-security/code-scanning/integrating-with-code-scanning/uploading-a-sarif-file-to-github#uploading-a-code-scanning-analysis-with-github-actions"
---

- [x] Usando la GitHub Action `github/codeql-action/upload-sarif`
- [ ] Al utilizar GitHub Actions los resultados SARIF se cargan automáticamente en GitHub.
- [ ] Solo puedes usar CodeQL al ejecutar el análisis de código en GitHub Actions. Las herramientas de análisis de código de terceros no son compatibles.
- [ ] Usando la GitHub Action `actions/upload-artifact`
