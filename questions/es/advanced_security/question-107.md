- [x] Es una función que permite a Dependabot descartar automáticamente las alertas de Dependabot que cumplan ciertos criterios.
- [ ] Las reglas de auto-triage se definen en el archivo de configuración `dependabot.yml` para especificar qué gestores de paquetes se deben usar para escanear tu proyecto en busca de vulnerabilidades.
- [ ] Las reglas de auto-triage de Dependabot se utilizan para eliminar automáticamente las dependencias antiguas de tu proyecto.
- [ ] Las reglas de auto-triage definen con qué frecuencia Dependabot debe escanear tu proyecto en busca de vulnerabilidades.
