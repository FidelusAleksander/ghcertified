documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-auto-triage-rules/about-dependabot-auto-triage-rules"

- [x] Es una función que permite a Dependabot descartar automáticamente alertas de Dependabot que cumplan ciertos criterios.
- [ ] Las reglas de auto-clasificación se definen en el archivo de configuración `dependabot.yml` para especificar qué administradores de paquetes deben usarse para escanear tu proyecto en busca de vulnerabilidades.
- [ ] Las reglas de auto-clasificación de Dependabot se usan para eliminar automáticamente dependencias antiguas en tu proyecto.
- [ ] Las reglas de auto-clasificación definen con qué frecuencia Dependabot debe escanear tu proyecto en busca de vulnerabilidades.
