---
question: "¿Para qué se utilizan las consultas de CodeQL?"
documentation: "https://codeql.github.com/docs/writing-codeql-queries/about-codeql-queries/"
---

- [x] Las consultas de CodeQL se pueden ejecutar contra una base de datos de CodeQL para identificar patrones que puedan indicar errores de codificación o vulnerabilidades de seguridad.
- [ ] Las consultas de CodeQL analizan tu base de código y se utilizan para crear una base de datos de CodeQL.
- [ ] Las consultas de CodeQL se utilizan para fines de revisión de código en GitHub.
- [ ] Las consultas de CodeQL son preguntas basadas en texto que puedes hacer al motor de CodeQL sobre tu base de código.
