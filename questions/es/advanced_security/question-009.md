---
question: "¿Cuál es el propósito del programa de socios para el escaneo de secretos?"
documentation: "https://docs.github.com/en/code-security/tutorials/secret-scanning-partner-program"
---

- [x] Los proveedores de servicios pueden asociarse con GitHub para que el formato de sus secretos pueda ser reconocido por el escaneo de secretos de GitHub.
- [ ] El programa de socios de GitHub permite a las empresas y organizaciones con licencia de GitHub Advanced Security utilizar el escaneo de secretos de GitHub para analizar sus repositorios.
- [ ] GitHub se asocia con empresas de seguridad externas para proporcionar escaneo de secretos para los repositorios de GitHub.
- [ ] Es un programa en el que profesionales de seguridad registrados pueden, de buena fe, informar a GitHub sobre cualquier secreto que encuentren en los repositorios de GitHub y recibir recompensas por ello.
