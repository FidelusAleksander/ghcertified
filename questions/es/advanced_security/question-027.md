---
question: "¿Cuándo se actualizará el gráfico de dependencias de GitHub para tu repositorio?"
documentation: "https://docs.github.com/en/code-security/supply-chain-security/understanding-your-software-supply-chain/about-supply-chain-security#what-is-the-dependency-graph"
---

- [x] Cuando alguien haga un push de un cambio al repositorio de una de tus dependencias.
- [x] Cuando hagas un push de un commit a la rama predeterminada del repositorio, únicamente si ese cambio añade o modifica un archivo de manifiesto/lock compatible.
- [ ] Cuando hagas un push de cualquier commit a la rama predeterminada del repositorio.
- [ ] Cuando tu repositorio publique una nueva versión.
- [ ] Cuando tu repositorio publique una nueva etiqueta git.
- [ ] Cuando se active el flujo de trabajo de GitHub Actions que utiliza la GitHub Action `actions/dependency-graph`.
