---
question: "¿Qué información proporcionan las alertas de Dependabot?"
documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-alerts/about-dependabot-alerts"
---

- [x] Las alertas de Dependabot te indican que tu repository utiliza un paquete que es inseguro.
- [ ] Las alertas de Dependabot te indican que tu repository está siendo utilizado por otros repositories públicos.
- [ ] Las alertas de Dependabot te indican que tu repository utiliza una versión no probada de un paquete.
- [ ] Las alertas de Dependabot te indican que tu repository utiliza una versión obsoleta de un paquete.
> Dependabot hace eso con las actualizaciones de versión, pero no con las alertas.
