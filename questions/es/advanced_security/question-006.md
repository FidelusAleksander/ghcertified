question: "¿Cuáles de estas funciones de seguridad de GitHub están disponibles de forma GRATUITA tanto para repositorios personales públicos como privados?"
documentation: "https://docs.github.com/en/code-security/getting-started/github-security-features"
---

- [x] Política de seguridad
- [x] Avisos de seguridad
- [x] Alertas y actualizaciones de seguridad de Dependabot
- [x] Actualizaciones de versión de Dependabot
- [ ] Escaneo de código con Dependabot
> Dependabot no es una herramienta para escaneo de código, es una herramienta para gestión de dependencias
- [ ] Escaneo de secretos con Dependabot
> Dependabot no es una herramienta para escaneo de secretos, es una herramienta para gestión de dependencias
- [ ] Escaneo de secretos
> El escaneo de secretos está disponible para repositorios públicos y para organizaciones con una licencia de GitHub Advanced Security. No está disponible para repositorios privados personales.
- [ ] Escaneo de código
> El escaneo de código está disponible para repositorios públicos y para organizaciones con una licencia de GitHub Advanced Security. No está disponible para repositorios privados personales.
