---
question: "¿Cómo puedes habilitar el análisis de código para un repositorio?"
documentation: "https://docs.github.com/en/code-security/code-scanning/enabling-code-scanning/configuring-default-setup-for-code-scanning"
---

- [x] Ve a la pestaña de seguridad en la configuración del repositorio y habilita el análisis de código con la configuración predeterminada o avanzada.
- [ ] Ve a la configuración de usuario y habilita el análisis de código; puedes elegir habilitarlo para todos o solo para algunos repositorios seleccionados.
- [ ] Agrega un archivo de configuración `.github/codeql.yml` al repositorio.
- [ ] Ve a la pestaña de seguridad en la configuración del repositorio y responde un cuestionario sobre el contenido del repositorio. Basado en las respuestas, GitHub habilitará el análisis de código con la configuración adecuada.
