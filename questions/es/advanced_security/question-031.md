---
question: "¿Cuáles de estas afirmaciones sobre Dependabot Alerts son verdaderas?"
documentation: "https://docs.github.com/en/code-security/concepts/supply-chain-security/dependabot-alerts"
---

- [x] Dependen parcialmente de la GitHub Advisory Database
- [x] Para habilitar Dependabot Alerts primero necesitas tener habilitado Dependency Graph en tu repositorio
- [x] Cuando GitHub identifica una dependencia vulnerable, genera una alerta de Dependabot y la muestra en la pestaña Security del repositorio
- [ ] Dependabot Alerts están habilitadas por defecto para todos los repositorios
- [ ] Dependabot Alerts están habilitadas por defecto para todos los repositorios públicos
- [ ] Dependabot alerts te informan que tu repositorio utiliza una versión desactualizada de un paquete
