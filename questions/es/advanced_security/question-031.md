---
question: "¿Cuáles de estas afirmaciones sobre Dependabot Alerts son verdaderas?"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/dependabot-alerts/about-dependabot-alerts"
---

- [x] Dependen parcialmente de la GitHub Advisory Database
- [x] Para habilitar Dependabot Alerts, primero necesitas tener activado el Dependency Graph en tu repositorio
- [x] Cuando GitHub identifica una dependencia vulnerable, genera una alerta de Dependabot y la muestra en la pestaña Security del repositorio
- [ ] Dependabot Alerts están habilitadas de forma predeterminada para todos los repositorios
- [ ] Dependabot Alerts están habilitadas de forma predeterminada para todos los repositorios públicos
- [ ] Las alertas de Dependabot te informan que tu repositorio utiliza una versión desactualizada de un paquete
