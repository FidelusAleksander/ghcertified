---
question: "¿Cuáles de estas afirmaciones sobre Dependabot Alerts son verdaderas?"
documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-alerts/about-dependabot-alerts"
---

- [x] Dependen parcialmente de la GitHub Advisory Database
- [x] Para habilitar Dependabot Alerts primero necesitas tener habilitado el Dependency Graph en tu repositorio
- [x] Cuando GitHub identifica una dependencia vulnerable, generan una alerta de Dependabot y la muestran en la pestaña de Seguridad del repositorio
- [ ] Dependabot Alerts están habilitados por defecto para todos los repositorios
- [ ] Dependabot Alerts están habilitados por defecto para todos los repositorios públicos
- [ ] Dependabot alerts informan que tu repositorio utiliza una versión desactualizada de un paquete
