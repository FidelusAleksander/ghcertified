---
question: "¿Cómo puedes establecer una política de seguridad predeterminada para todos los repositorios en la Organización de GitHub `my-org`?"
documentation: "https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file#supported-file-types"
---

- [x] Creando un archivo `SECURITY.md` en el repositorio `my-org/.github`
- [ ] Editando la política de seguridad en la configuración de `Code Security and analysis` de la organización
- [ ] Las políticas de seguridad predeterminadas solo pueden ser establecidas por el soporte de GitHub
- [ ] Puedes establecer una política de seguridad predeterminada para todos los repositorios en la Organización de GitHub `my-org` añadiendo un archivo `SECURITY.md` a cada repositorio individual.
