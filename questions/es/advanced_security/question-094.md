question: "¿Qué es una política de seguridad de GitHub?"
documentation: "https://docs.github.com/en/code-security/getting-started/adding-a-security-policy-to-your-repository#about-security-policies"
---

- [x] Es un documento que instruye a los usuarios sobre cómo informar de manera responsable las vulnerabilidades de seguridad en un proyecto. Normalmente se define en un archivo `SECURITY.md` en un repository.
- [ ] Es una herramienta para solucionar automáticamente las vulnerabilidades de seguridad en tu código.
- [ ] Es una característica que te permite encriptar tu repository.
- [ ] Una política de seguridad de GitHub es un servicio de suscripción que proporciona protección antivirus para tus proyectos.
