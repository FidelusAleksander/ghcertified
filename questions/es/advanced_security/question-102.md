---
question: "¿Quién puede habilitar las alertas de Dependabot en un repositorio?"
documentation: "https://docs.github.com/en/code-security/concepts/supply-chain-security/dependabot-alerts"
---

- [x] Los propietarios del repositorio y las personas con acceso de administrador
- [ ] Solo el propietario del repositorio
- [ ] Las alertas de Dependabot están habilitadas en todos los repositorios por GitHub y no pueden ser deshabilitadas ni habilitadas por ningún individuo.
- [ ] Las alertas de Dependabot se habilitan agregando una GitHub Action al repositorio, por lo que cualquiera con acceso de escritura al repositorio puede habilitarlas.
