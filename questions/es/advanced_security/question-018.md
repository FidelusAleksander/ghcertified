---
question: "Cuando GitHub ejecuta un análisis de escaneo de secretos de todo el código histórico en los repositorios de la empresa, ¿cuál es el comportamiento de notificación?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning/monitoring-alerts#historical-scans"
---

- [x] GitHub notifica a los propietarios de la empresa y a los gestores de seguridad, incluso si no se encuentran secretos.
- [x] GitHub notifica a los administradores del Repository, gestores de seguridad y usuarios con roles personalizados con acceso de lectura/escritura cada vez que se detecta un secreto en un repositorio.
- [ ] GitHub notifica a los propietarios de la empresa y a los gestores de seguridad, solo si detecta secretos expuestos.
- [ ] GitHub notifica a los autores de los commits que contienen secretos expuestos.
