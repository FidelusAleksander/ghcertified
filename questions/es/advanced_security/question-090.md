documentation: "https://docs.github.com/en/code-security/code-scanning/integrating-with-code-scanning/sarif-support-for-code-scanning#uploading-more-than-one-sarif-file-for-a-commit"
---

- [x] Usar la categoría para distinguir entre múltiples análisis para la misma herramienta o commit, pero realizados en diferentes lenguajes o diferentes partes del código.
- [ ] Usar la categoría para distinguir archivos que han sido analizados de archivos que no han sido analizados.
- [ ] Usar la categoría para distinguir archivos que contienen vulnerabilidades de archivos que no contienen vulnerabilidades.
- [ ] Usar una categoría diferente para cada archivo que ha sido analizado para rastrear fácilmente las vulnerabilidades hasta los archivos que las contienen.
