- [x] 500 para organizaciones/empresas y 100 para repositorios.
- [ ] 100 para organizaciones/empresas y 500 para repositorios.
- [ ] 100 para organizaciones, empresas y repositorios.
- [ ] No hay límite para el número de patrones personalizados que puedes definir para el escaneo de secretos en GitHub.
