---
question: "Al usar un sistema de CI de terceros para ejecutar el análisis de código, ¿qué herramienta de GitHub necesitas para analizar la base de código?"
documentation: "https://docs.github.com/en/code-security/how-tos/find-and-fix-code-vulnerabilities/integrate-with-existing-tools/use-with-existing-ci-system#about-using-code-scanning-with-your-existing-ci-system"
---

- [x] No necesitas específicamente una herramienta de GitHub, cualquier herramienta de análisis estático que pueda producir resultados en formato SARIF funcionará.
- [ ] Necesitas instalar la herramienta GitHub Code Scanning.
- [ ] Necesitas instalar CodeQL CLI.
- [ ] Necesitas instalar GitHub CLI.
