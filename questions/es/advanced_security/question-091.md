question: "¿Cómo puedes habilitar las funciones de GitHub Advanced Security en GitHub Enterprise Server?"

- [x] En la pestaña de Seguridad de la consola de administración del sitio.
- [x] Conectándote directamente a la instancia de GitHub Enterprise Server mediante SSH y utilizando los comandos del shell administrativo `ghe-config`.
- [ ] Solicitando una actualización al soporte de GitHub.
- [ ] Configurando la opción `github.advanced_security.enabled` a `true` en el archivo `config.yml` del directorio `/etc/github` en la instancia de GitHub Enterprise Server.
- [ ] Configurando la opción `github.advanced_security.enabled` a `true` en el archivo `config.yml` del repositorio `.github`.
