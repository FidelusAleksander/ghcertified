- [x] El escaneo de secretos analiza tu repositorio en busca de secretos como claves privadas o tokens.
- [ ] El escaneo de secretos analiza tu repositorio en busca de posibles vulnerabilidades de código que puedan exponer secretos como claves privadas o tokens.
- [ ] El escaneo de secretos es una herramienta para el almacenamiento y gestión segura de secretos.
- [ ] El escaneo de secretos es un gancho de git que escanea tus commits en busca de secretos como claves privadas o tokens antes de que se envíen a GitHub.
