- [x] El escaneo de secretos examina tu repository en busca de secretos como claves privadas o tokens.
- [ ] El escaneo de secretos examina tu repository en busca de posibles vulnerabilidades de código que puedan exponer secretos como claves privadas o tokens.
- [ ] El escaneo de secretos es una herramienta para el almacenamiento y gestión seguros de secretos.
- [ ] El escaneo de secretos es un git hook que escaneará tus commits en busca de secretos como claves privadas o tokens antes de que se suban a GitHub.
