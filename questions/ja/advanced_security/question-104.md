---
question: "組織内のすべてのリポジトリでDependabot Alertsを有効にするには、次の操作を行います。"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/secure-your-dependencies/configure-dependabot-alerts"
---

- [x] 組織の`Code security and analysis`設定に移動し、すべてのリポジトリに対してDependabot Alertsを一括で有効にします。
- [ ] 組織内のすべてのリポジトリをプライベートにします。
- [ ] 組織内のすべてのリポジトリで、`alerts`パラメータを`true`に設定して`actions/enable-ghas` GitHub Actionを実行します。
- [ ] 組織内のすべてのリポジトリでDependabot Alertsを有効にするスクリプトを作成します。
