- [x] 組織の`Code security and analysis`設定に移動し、すべてのリポジトリに対してDependabot Security Updatesを一括で有効にします。
- [ ] 組織内のすべてのリポジトリをプライベートにします。
- [ ] 組織内のすべてのリポジトリで、`security-updates`パラメータを`true`に設定した`actions/enable-ghas` GitHub Actionを実行します。
- [ ] 組織内のすべてのリポジトリでDependabot Security Updatesを有効にするスクリプトを作成します。
