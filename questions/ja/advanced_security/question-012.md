---
question: "GitHub 秘密スキャンパートナープログラムについて正しいものは次のうちどれですか？"
documentation: "https://docs.github.com/en/code-security/tutorials/secret-scanning-partner-program"
---

- [x] これは、サービスプロバイダーが自社が発行する秘密の正規表現パターンを GitHub に提供することで、GitHub の秘密スキャンがそれらを認識できるようにするプログラムです。
- [x] GitHub が提携サービスプロバイダーの秘密を特定すると、漏洩した秘密についてサービスプロバイダーに通知します。
- [x] パートナーは、GitHub から漏洩した秘密に関する通知を受け取った場合、秘密の無効化や漏洩した秘密の所有者に通知するなどのアクションを実行できます。
- [ ] これは、サービスプロバイダーが GitHub リポジトリ内の秘密を特定するために、秘密 GitHub スキャン API にアクセスできる権限を付与します。
> 秘密スキャンは常に GitHub が実行し、パートナーが実行することはありません。
- [ ] GitHub は漏洩した秘密を自動的に無効化し、それらが GitHub によって無効化されたことをサービスプロバイダーに通知することができます。
> GitHub は漏洩した秘密の無効化には関与せず、サービスプロバイダーがそれを実行するオプションを持ちます。
