question: "これらのGitHubセキュリティ機能のうち、パブリックまたはプライベートの個人リポジトリの両方で無料で利用可能なのはどれですか？"
documentation: "https://docs.github.com/en/code-security/getting-started/github-security-features"

- [x] セキュリティポリシー
- [x] セキュリティ勧告
- [x] Dependabotアラートとセキュリティ更新
- [x] Dependabotバージョン更新
- [ ] Dependabotコードスキャン
> Dependabotはコードスキャンツールではなく、依存関係管理ツールです。
- [ ] Dependabotシークレットスキャン
> Dependabotはシークレットスキャンツールではなく、依存関係管理ツールです。
- [ ] シークレットスキャン
> シークレットスキャンは、パブリックリポジトリやGitHub Advanced Securityライセンスを持つ組織で利用可能です。個人のプライベートリポジトリでは利用できません。
- [ ] コードスキャン
> コードスキャンは、パブリックリポジトリやGitHub Advanced Securityライセンスを持つ組織で利用可能です。個人のプライベートリポジトリでは利用できません。
