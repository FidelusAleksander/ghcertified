- [x] カスタム設定ファイルを使用し、そこで追加のクエリを定義する
- [x] CodeQL分析のGitHub Actionsワークフロー内で、`github/codeql-action/init`アクションへの入力パラメーターとしてカスタマイズを定義する
- [ ] CodeQL CLIを使用してカスタム設定ファイルで分析を実行する
- [ ] Security / Code scanningリポジトリ設定内でカスタマイズを定義する
- [ ] `github/codeql-customizations` GitHub Actionを使用する
