documentation: "https://docs.github.com/en/code-security/reference/code-scanning/workflow-configuration-options"

- [x] カスタム構成ファイルを使用し、そこに追加のクエリを定義する
- [x] CodeQL分析のGitHub Actionsワークフロー内で、`github/codeql-action/init`アクションへの入力パラメータとしてカスタマイズを定義する
- [ ] カスタム構成ファイルを使用してCodeQL CLIを用いて分析を実行する
- [ ] Security / Code scanningリポジトリ設定内でカスタマイズを定義する
- [ ] `github/codeql-customizations` GitHubアクションを使用する
