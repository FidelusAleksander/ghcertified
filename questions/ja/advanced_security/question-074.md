---
question: "次のうち、CodeQL スキャンのデフォルトセットアップを有効にすることに関する記述で正しいものはどれですか？"
documentation: "https://docs.github.com/ja/code-security/code-scanning/enabling-code-scanning/configuring-default-setup-for-code-scanning"
---

- [x] 組織設定で、組織内のすべての適格なリポジトリに対してデフォルトセットアップを一括で有効にすることができる
- [x] 事前条件として GitHub Actions を有効にする必要がある
- [x] リポジトリの内容にかかわらず、任意のリポジトリでデフォルトセットアップを有効にすることができる
- [ ] 少なくとも 1 つの CodeQL 対応言語を含むリポジトリでのみデフォルトセットアップを有効にすることができる
> CodeQL 対応言語を含まないリポジトリでデフォルトセットアップを有効にした場合、デフォルトセットアップはスキャンを実行しない
- [ ] デフォルトセットアップは、構成可能なスケジュールに基づいてリポジトリをスキャンする。イベントベースのスキャンを行うには、GitHub Action ワークフローを構成する必要がある
> デフォルトセットアップには、スケジュールスキャンと、デフォルトおよび保護されたブランチにおけるプルリクエスト／プッシュスキャンが含まれる
- [ ] デフォルトの CodeQL スキャンセットアップでは、デフォルトのクエリスイートのみを使用できる
