---
question: "サードパーティ製のCIシステムを使用してコードスキャンを実行する場合、コードベースを分析するために必要なGitHubツールは何ですか？"
documentation: "https://docs.github.com/en/code-security/code-scanning/integrating-with-code-scanning/using-code-scanning-with-your-existing-ci-system#about-using-code-scanning-with-your-existing-ci-system"
---

- [x] 特にGitHubツールは必要ありません。SARIF形式で結果を生成できる任意の静的解析ツールが使用できます。
- [ ] GitHub Code Scanningツールをインストールする必要があります。
- [ ] CodeQL CLIをインストールする必要があります。
- [ ] GitHub CLIをインストールする必要があります。
