- [x] GitHub は Git をコア技術として使用し、コラボレーションとコード管理を容易にする機能を追加したプラットフォームです。
- [ ] GitHub は、プロジェクトの共同作業を容易にする機能を備えた分散型バージョン管理システムです。
- [ ] GitHub は、集中型バージョン管理を使用してプロジェクトの共同作業を容易にする機能を備えたファイル共有プラットフォームです。
- [ ] GitHub は、迅速な製品開発を促進する機能を備えた最新の CI/CD プラットフォームです。
