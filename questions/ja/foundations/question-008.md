---
question: "次のワークフローのうち、GitHub Projects に組み込まれている自動化機能はどれですか？"
documentation: "https://docs.github.com/en/issues/planning-and-tracking-with-projects/automating-your-project/using-the-built-in-automations"
---

- [x] プロジェクト内の Issue または Pull Request がクローズされると、そのステータスが Done に設定される
- [x] プロジェクト内の Pull Request がマージされると、そのステータスが Done に設定される
- [ ] プロジェクト内の Issue または Pull Request が作成されると、そのステータスが Done に設定される
- [ ] 新しいコントリビューターがリポジトリに追加されると、未割り当ての Issue がその人に割り当てられる
- [ ] GitHub Action がトリガーされると、GitHub Project に新しいアイテムが作成される
