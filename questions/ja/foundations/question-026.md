---
question: "GitHub Codespaceとは何ですか？"
documentation: "https://docs.github.com/en/codespaces/about-codespaces/what-are-codespaces"
---

- [x] リポジトリ専用に設定された事前構成済みの開発環境です。ローカル開発環境をセットアップせずに、プロジェクト用のコードを書き始めることができます。
- [ ] GitHub Codespaceは特別なハードウェアを必要とするインタラクティブなホログラフィックコーディング環境です。
- [ ] GitHub Codespaceは、ほぼ完璧なコードを自動生成するAI駆動のコーディングツールです。
- [ ] GitHub Codespaceは、開発者が会合してコーディングに関するトピックを議論できる、毎年開催されるGitHub主催のイベントです。
