---
question: "GitHub Projects と GitHub Projects classic の違いは何ですか？"
documentation: "https://docs.github.com/en/issues/planning-and-tracking-with-projects"
---

- [x] GitHub Projects は、新しい GitHub のエクスペリエンスであり、従来の GitHub Projects classic よりも多くの新機能と改善を提供します。
- [ ] GitHub Projects classic は、新しい GitHub のエクスペリエンスであり、従来の GitHub Projects よりも多くの新機能と改善を提供します。
- [ ] GitHub Projects classic は、GitHub Projects を管理するよりシンプルな方法で、初心者に推奨されます。
- [ ] GitHub Projects classic は、GitHub Projects を強化したバージョンであり、ロードマップやスプリントなどのあらかじめ用意されたテンプレートを提供します。
