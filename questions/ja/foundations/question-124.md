- [x] ディスカッションをピン留めする  
- [x] 別のリポジトリのディスカッションに転送する（管理者権限がある場合）  
- [ ] ディスカッションをシークレットGistに変換する  
- [x] ディスカッションをIssueに変換する  
- [ ] 既存のPull Requestにマージする
