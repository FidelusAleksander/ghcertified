- [x] 既存のリポジトリからリポジトリテンプレートを作成する
- [x] リポジトリテンプレートから新しいリポジトリを作成する
- [ ] リポジトリテンプレートから組織を作成する
- [ ] リポジトリテンプレートから作成されたすべてのリポジトリを削除する
- [ ] 組織が提供するリポジトリテンプレートを使用していないすべてのリポジトリを削除する
