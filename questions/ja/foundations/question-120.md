- [x] 組織全体で許可またはブロックされるアクションを設定できる。  
- [ ] 90日以上前のプライベートリポジトリのすべてのワークフローランを削除できる。  
- [ ] 組織内のリポジトリのコードレビューを無効にできる。  
- [ ] `Require status checks to pass before merging`のブランチ保護ルールを削除できる。
