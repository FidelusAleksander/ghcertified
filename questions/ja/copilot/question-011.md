---
question: "リポジトリに適した回答を生成するために、GitHub Copilot に文脈をどのように提供できますか？"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/copilot/customizing-copilot/adding-custom-instructions-for-github-copilot"
---

- [x] リポジトリ内に `.github/copilot-instructions.md` という名前のファイルを作成する
- [ ] プロジェクトの詳細を記載したメールを GitHub サポートに送信する
- [ ] カスタム指示を含むように `.gitconfig` ファイルを変更する
> `.gitconfig` ファイルを変更しても、GitHub Copilot にカスタム指示を提供することはできません。
- [ ] 必要な文脈を記載した `copilot-instructions` という名前の GitHub Issue をリポジトリに作成する
> GitHub Issue を作成しても、GitHub Copilot にカスタム指示を提供することはできません。
