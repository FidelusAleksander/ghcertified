---
question: "リポジトリにカスタマイズされた応答を生成するために、GitHub Copilot にコンテキストを提供するにはどうすればよいですか？"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/copilot/how-tos/copilot-on-github/customize-copilot/add-custom-instructions/add-repository-instructions"
---

- [x] リポジトリに `.github/copilot-instructions.md` という名前のファイルを作成する
- [ ] プロジェクトの詳細を記載したメールを GitHub サポートに送信する
- [ ] `.gitconfig` ファイルを修正してカスタム指示を追加する
> `.gitconfig` ファイルを修正しても、GitHub Copilot にカスタム指示を提供することはできません。
- [ ] 必要なコンテキストを記載した `copilot-instructions` という名前の GitHub Issue をリポジトリに作成する
> GitHub Issue を作成しても、GitHub Copilot にカスタム指示を提供することはできません。
