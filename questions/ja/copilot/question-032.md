- [x] チャット参加者（例: `@workspace` や `@vscode`）を使用して、コードベース、特定の分野や技術に関する追加のコンテキストを提供できます。
- [x] スラッシュコマンド（例: `/tests`、`/fix`、`/explain`）を使用すると、プロンプトで達成したいことを簡潔に伝えることができます。
- [ ] チャット参加者、コマンド、およびチャット変数を1つのプロンプトで組み合わせることはできません。
- [ ] チャット参加者は、同じOrganization内の他のメンバーをタグ付けするためだけに使用できます。
> チャット参加者（例: `@workspace` や `@vscode`）は、コードベース、分野、または技術に関するコンテキストを提供するために設計されており、他の人をタグ付けするためのものではありません。
