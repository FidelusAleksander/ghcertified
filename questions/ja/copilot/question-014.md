---
question: "GitHub Copilotを始める最も簡単な方法は何ですか？"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [ ] GitHub Supportにアクセスをリクエストし、GitHub Copilotを使用する前に承認を待つ。
- [ ] Copilotウェブサイトを使用し、提案を求める際にコードを貼り付ける。
- [x] Visual Studio Codeなど、お好みの環境にCopilot拡張機能をインストールする。
- [ ] 新しいパブリックGitHubリポジトリを作成し、Copilotを有効にしてコードをスキャンし、提案を受け取る。
