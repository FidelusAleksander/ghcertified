---
question: "GitHub Copilot を始める最も簡単な方法は何ですか？"
documentation: "https://docs.github.com/en/copilot/using-github-copilot/getting-started-with-github-copilot"
---

- [ ] GitHub Support にアクセスをリクエストし、GitHub Copilot を使用する前に承認を待ちます。
- [ ] Copilot のウェブサイトを使用し、コードを貼り付けて提案を依頼します。
- [x] Visual Studio Code のような好みの環境に Copilot 拡張機能をインストールします。
- [ ] 新しいパブリック GitHub リポジトリを作成し、Copilot を有効にしてコードをスキャンし、提案を受け取ります。
