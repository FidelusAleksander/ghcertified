---
question: "ワークフローについて正しいのはどれですか？"
documentation: "https://docs.github.com/en/actions/using-workflows/about-workflows"
---

- [x] ワークフローは一度に1つまたは複数のジョブを実行できます
- [x] ワークフローは手動で、イベントによって、またはスケジュールに従ってトリガーできます
- [x] ワークフローは `.github/workflows` ディレクトリ内で定義する必要があります
- [ ] ワークフローはスケジュールでのみ実行できます
- [ ] ワークフローは一度に1つのジョブしか実行できません
- [ ] ワークフローは `.yaml`、`.json`、`.toml` のいずれの形式でも記述できます
> ワークフローは `.yaml` 形式でのみ定義できます
- [ ] ワークフローは GitHub Marketplace で共有できます
> Actions（ワークフローではなく）は GitHub Marketplace で共有できます
