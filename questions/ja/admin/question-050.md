- [x] 組織の請求設定を管理し、支払い方法の更新や請求書の確認を含む。
- [ ] チームの権限やリポジトリの設定を管理することができる。
- [ ] 組織内のすべてのリポジトリを作成および削除する。
- [ ] 組織の所有者を割り当てたり削除したりする。
