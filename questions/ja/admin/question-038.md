- [x] `Site admin` ダッシュボードに移動し、`Enterprise account` 設定でライセンス使用状況を確認します。
- [ ] GitHub Supportへ直接メールでリクエストを送信します。
- [ ] 組織の公開プロファイルページでリアルタイムの統計を確認します。
- [ ] GitHub APIを使用して、GitHub Enterprise Serverのライセンス使用状況データをリアルタイムで取得します。
