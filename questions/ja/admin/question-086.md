---
question: "GitHubリポジトリにセルフホストランナーを追加するにはどうすればよいですか？"
documentation: "https://docs.github.com/en/actions/how-tos/manage-runners/self-hosted-runners/add-runners"
---

- [x] リポジトリ設定に移動し、Actionsを選択してRunnersを選択し、セルフホストランナーアプリケーションをダウンロード、構成、および実行する手順に従います。
- [ ] リポジトリをローカルマシンに直接クローンし、特定のスクリプトを実行してセルフホストランナーとして自動的に追加します。
- [ ] GitHubサポートにメールを送り、リポジトリにセルフホストランナーを追加するよう依頼します。
- [ ] GitHub APIを使用して、手動での構成なしにセルフホストランナーを追加するリクエストを送信します。
