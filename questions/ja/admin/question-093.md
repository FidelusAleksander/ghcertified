- [x] 適切なスコープを持つ PAT を使用して認証する
- [ ] GitHub Packages は認証なしで公開され、共有可能です。
- [ ] GitHub Packages へのアクセスは GitHub Enterprise アカウントに限定されています。
- [ ] 特別な認証トークンを必要とせず、デフォルトの GitHub ログインを使用してアクセスします。
