- [x] リポジトリの依存関係および依存先の表現です。
- [ ] 依存関係グラフというものは存在しません。
- [ ] リポジトリ内の依存関係のバージョン更新を自動的に提案するツールです。
- [ ] オープンソースソフトウェアパッケージにおける既知の脆弱性をまとめたGitHub管理のリストです。
