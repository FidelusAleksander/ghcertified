- [x] リポジトリの依存関係と依存元を表すものです。
- [ ] 依存関係グラフというものは存在しません。
- [ ] リポジトリ内の依存関係のバージョン更新を自動的に提案するツールです。
- [ ] オープンソースソフトウェアパッケージの既知の脆弱性をGitHubが管理するリストです。
