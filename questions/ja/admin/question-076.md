---
question: "GitHub Apps と OAuth apps の違いは何ですか？"
documentation: "https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/differences-between-github-apps-and-oauth-apps"
---

- [x] GitHub Apps は特定のリポジトリに対して詳細な権限を提供しますが、OAuth Apps はすべてのリポジトリにわたるユーザーデータへのアクセスを要求します。
- [ ] GitHub Apps と OAuth Apps に違いはなく、同じレベルの権限を提供します。
- [ ] OAuth Apps は GitHub Apps よりも詳細な権限を提供します。
- [ ] GitHub Apps は公開リポジトリのみアクセスできますが、OAuth Apps は公開リポジトリとプライベートリポジトリの両方にアクセスできます。
