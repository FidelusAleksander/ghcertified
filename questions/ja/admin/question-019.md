- [x] ユーザーは GitHub.com のリソースにアクセスするために、会社の IdP で認証する必要があります。
- [ ] ユーザーは企業の関与なしに公開 GitHub ログインページを通じて認証します。
- [ ] 企業管理ユーザーに対してアクセスの容易さのために認証がバイパスされます。
- [ ] ユーザーは特別な GitHub 管理の認証トークンを提供します。
