- [x] ビジネスおよびオープンソースプロジェクトが、複数のプロジェクトを対象に高度なセキュリティおよび管理機能を使用して共同作業することを可能にする。
- [ ] 個々のユーザーに自分のプライベートプロジェクトをホストするための個人的なスペースを提供する。
- [ ] コードリポジトリのバックアップサービスとして機能する。
- [ ] 個人ブログやウェブサイトホスティングのプラットフォームを提供する。
