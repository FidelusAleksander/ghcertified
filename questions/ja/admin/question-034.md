- [x] 災害復旧計画を立て、バックアップを構成する
- [ ] サードパーティのクラウドストレージサービスを使用してバックアップを行う
- [ ] GitHub の自動クラウドバックアップサービスに依存する
- [ ] 定期的にデータをローカルストレージにエクスポートして手動でバックアップする
