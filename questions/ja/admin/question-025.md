---
question: "GitHub Enterprise Cloud と GitHub Enterprise Server の違いは何ですか？"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/overview/about-github-for-enterprises#about-deployment-options"
---

- [x] GitHub Enterprise Cloud は GitHub によってホスティングされ、GitHub Enterprise Server は自身のサーバーまたはクラウドプロバイダー上でホスティングされます。
- [ ] GitHub Enterprise Cloud は自身のサーバー上でホスティングされ、GitHub Enterprise Server は GitHub によってホスティングされます。
- [ ] GitHub Enterprise Cloud は公開リポジトリのみを提供し、GitHub Enterprise Server は非公開リポジトリを提供します。
- [ ] GitHub Enterprise Cloud は個人利用向けに設計されており、GitHub Enterprise Server は組織向けです。
