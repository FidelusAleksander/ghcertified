---
question: "パブリックリポジトリにおけるGitHub Actionsの利用料金は？"
documentation: "https://docs.github.com/en/billing/concepts/product-billing/github-actions"
---

- [x] パブリックリポジトリでのGitHub Actionsの利用は無料です。
- [ ] パブリックリポジトリで利用した分だけGitHub Actionsに課金されます。
- [ ] パブリックリポジトリにはプライベートリポジトリの半分の無料分しか提供されません。
- [ ] GitHub Actionsはパブリックリポジトリでは利用できません。
