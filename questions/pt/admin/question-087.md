---
question: "O que é necessário para adicionar um executor hospedado por conta própria a uma organização no GitHub?"
documentation: "https://docs.github.com/en/actions/how-tos/manage-runners/self-hosted-runners/add-runners"
---

- [x] Acesso de proprietário da organização, navegar para as configurações da organização, selecionar Actions, depois Runners, e seguir as instruções de configuração para o executor hospedado por conta própria.
- [ ] Acesso geral de membro sem permissões especiais, e utilizar uma GitHub Action para registrar automaticamente um executor hospedado por conta própria.
- [ ] Acesso a repositórios públicos dentro da organização e execução manual de script no servidor destinado como executor.
- [ ] Enviar um formulário no site do GitHub para solicitar à equipe do GitHub a adição manual de um executor hospedado por conta própria à sua organização.
