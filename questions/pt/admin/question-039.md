---
question: "Como um administrador de organização pode definir permissões padrão para novos membros em uma organização no GitHub?"
documentation: "https://docs.github.com/en/organizations/managing-user-access-to-your-organizations-repositories/managing-repository-roles/setting-base-permissions-for-an-organization#setting-base-permissions"
---

- [x] Na seção `Access` das configurações da organização, selecione `Member privileges`, depois, em `Base permissions`, escolha um nível de permissão e confirme a alteração.
- [ ] As permissões padrão são determinadas automaticamente pelo GitHub com base na atividade da organização e não podem ser definidas manualmente.
- [ ] Configurando permissões individualmente para cada novo membro à medida que são adicionados à organização, sem um nível padrão disponível.
- [ ] As permissões padrão para novos membros são definidas através de uma chamada à API do GitHub especificando o nível de permissão base desejado para a organização.
