question: "Como você pode garantir que os seus runners auto-hospedados ou maiores runners hospedados no GitHub possam se comunicar com o GitHub ao usar uma lista de permissão de IP?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/configuration/hardening-security-for-your-enterprise/restricting-network-traffic-to-your-enterprise-with-an-ip-allow-list#using-github-actions-with-an-ip-allow-list"
---

- [x] Adicione o endereço IP ou o intervalo de endereços IP dos seus runners à lista de permissão de IP configurada para a sua empresa
- [ ] Desative a lista de permissão de IP para os runners do GitHub Actions
- [ ] Registre os runners em Enterprise > Network security > Actions settings
- [ ] Use uma VPN para todos os runners para contornar a lista de permissão de IP
