- [ ] Exigindo que todos os membros utilizem autenticação biométrica para acessar o GitHub.
- [ ] Tornando todos os repositórios dentro da organização públicos para garantir transparência.
- [ ] Desativando a criação e o processamento de issues e pull requests para reduzir superfícies de ataque.
