question: "Qual destas melhor define as Políticas de GitHub Enterprise?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/enforcing-policies/enforcing-policies-for-your-enterprise/enforcing-repository-management-policies-in-your-enterprise"
---

- [x] Configurações gerenciadas por proprietários da empresa para controlar aspectos como gerenciamento de repositórios, acesso a equipes e recursos de segurança dentro de suas organizações GitHub.
- [ ] Políticas definidas no nível empresarial que ditam os detalhes de faturamento e assinatura para todas as organizações em uma conta corporativa.
- [ ] Diretrizes sugeridas pelo GitHub para melhorar a colaboração e produtividade em todas as organizações sem capacidades de imposição.
- [ ] Configurações padrão aplicadas a todos os novos repositórios dentro de uma organização, que podem ser substituídas pelos proprietários da organização.
