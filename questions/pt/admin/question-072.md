- [x] Políticas corporativas determinam uma política uniforme em todas as organizações dentro de uma empresa.
- [ ] Configurações individuais de usuários que se aplicam apenas a contas pessoais no GitHub Enterprise Cloud, sem relação com governança organizacional ou empresarial.
- [ ] Uma coleção de práticas recomendadas para projetos open-source hospedados no GitHub, voltadas para melhorar o engajamento da comunidade e as contribuições.
- [ ] Protocolos de segurança que automaticamente verificam e corrigem vulnerabilidades nos códigos de todos os repositórios dentro da conta corporativa.
