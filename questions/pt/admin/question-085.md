---
question: "Como você deve selecionar runners apropriados para atender às cargas de trabalho específicas do seu workflow?"
documentation: "https://docs.github.com/en/actions/hosting-your-own-runners/managing-self-hosted-runners/about-self-hosted-runners#differences-between-github-hosted-and-self-hosted-runners"
---

- [x] Escolha runners self-hosted para ambientes personalizados ou GitHub-hosted para atualizações automáticas e instâncias limpas.
- [ ] Sempre prefira runners GitHub-hosted para todas as cargas de trabalho para garantir segurança máxima.
- [ ] Use apenas runners self-hosted, independentemente da complexidade da carga de trabalho, para minimizar custos.
- [ ] Restrinja-se a usar runners GitHub-hosted com configurações de hardware personalizadas para melhor desempenho.
