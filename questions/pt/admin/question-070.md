---
question: "Quais dessas práticas podem ajudar a evitar o envio acidental de dados ou arquivos sensíveis para um repositório Git?"
documentation: "https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/removing-sensitive-data-from-a-repository#avoiding-accidental-commits-in-the-future"
---

- [x] Adicionar arquivos ou padrões sensíveis a um arquivo `.gitignore`
- [ ] Alterar suas senhas regularmente
- [ ] Desabilitar o acesso de push ao repositório
- [ ] Usar `git add .` antes de todos os commits
- [ ] Desabilitar todas as regras `.gitignore` para garantir que todos os arquivos sejam rastreados
