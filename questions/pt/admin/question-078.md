- [x] Usando webhooks para se inscrever em informações sobre trabalhos e execuções de workflows, e potencialmente utilizando um sistema de arquivamento de dados.
- [x] Utilizando métricas de uso e desempenho do GitHub Actions disponíveis nos níveis da organização e do repositório.
> As métricas do GitHub Actions são acessíveis na aba Insights nos níveis da organização e do repositório, fornecendo dados de uso e desempenho como tempos de execução, tempos de fila e taxas de falha.
- [ ] Contando manualmente as execuções e os trabalhos de workflows em cada repositório no final de cada mês.
- [ ] O uso do GitHub Actions não pode ser rastreado; as empresas devem estimar o uso com base no feedback dos desenvolvedores.
