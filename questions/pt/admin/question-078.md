- [x] Usando webhooks para se inscrever em informações sobre jobs e execuções de workflows, e potencialmente utilizando um sistema de arquivamento de dados.
- [ ] Contando manualmente as execuções de workflows e jobs em cada repositório ao final de cada mês.
- [ ] Através de acesso ilimitado ao painel de análise interna do GitHub sem qualquer configuração adicional.
- [ ] O uso do GitHub Actions não pode ser rastreado; as empresas devem estimar o uso com base no feedback dos desenvolvedores.
