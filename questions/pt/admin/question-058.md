---
question: "Qual é o papel de um Security Manager dentro de uma organização do GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] Um Security Manager tem permissões para visualizar alertas de segurança e gerenciar configurações de segurança de código em toda a organização, além de permissões de leitura para todos os repositórios.
- [ ] Security Managers são responsáveis por transações financeiras e segurança das informações de faturamento dentro da organização.
- [ ] Eles podem excluir ou transferir a propriedade de qualquer repositório dentro da organização por razões de segurança.
- [ ] Security Managers gerenciam exclusivamente as configurações de autenticação em dois fatores para todos os membros da organização.
