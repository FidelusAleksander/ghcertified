---
question: "Como o GitHub Enterprise Cloud difere do GitHub Enterprise Server?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/overview/about-github-for-enterprises#about-deployment-options"
---

- [x] O GitHub Enterprise Cloud é hospedado pelo GitHub, enquanto o GitHub Enterprise Server é hospedado em seus próprios servidores ou em um provedor de nuvem.
- [ ] O GitHub Enterprise Cloud é hospedado em seus servidores, enquanto o GitHub Enterprise Server é hospedado pelo GitHub
- [ ] O GitHub Enterprise Cloud oferece apenas repositórios públicos, enquanto o GitHub Enterprise Server oferece repositórios privados
- [ ] O GitHub Enterprise Cloud é projetado para uso individual, enquanto o GitHub Enterprise Server é para organizações
