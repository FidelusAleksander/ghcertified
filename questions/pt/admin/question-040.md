- [x] Permite o reflexo automático de alterações de membros no grupo do IdP no GitHub, reduzindo a necessidade de atualizações manuais e scripts personalizados ao sincronizar equipes nas organizações do GitHub com grupos do IdP.
- [ ] Permite que o GitHub funcione como um provedor de identidade para o Microsoft Entra ID, simplificando os processos de autenticação de usuários.
- [ ] Converte automaticamente as associações de grupos do Microsoft Entra ID em repositórios correspondentes no GitHub, facilitando a alocação de recursos.
- [ ] Faz backup dos dados da organização do GitHub nos servidores do Microsoft Entra ID, fornecendo uma camada adicional de redundância de dados.
