---
question: "Quais etapas os administradores da organização devem seguir para acessar os registros de auditoria das ações realizadas dentro da organização no GitHub?"
documentation: "https://docs.github.com/en/organizations/keeping-your-organization-secure/managing-security-settings-for-your-organization/reviewing-the-audit-log-for-your-organization"
---

- [x] Vá para Configurações da Organização > Arquivo > Logs > Registro de auditoria. Em seguida, use o menu Exportar para exportar como JSON ou CSV.
- [ ] Envie um e-mail diretamente para o suporte do GitHub solicitando que um registro de auditoria seja enviado mensalmente para o e-mail do administrador da organização.
- [ ] Navegue até a página principal do GitHub, use a barra de pesquisa para digitar "Registros de auditoria da organização" e aguarde que o GitHub baixe o registro automaticamente.
- [ ] Verifique os perfis individuais dos membros para obter um resumo de suas ações, compile esses resumos manualmente e os formate para auditoria.
