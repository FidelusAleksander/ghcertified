---
question: "Como você pode conceder a um usuário as permissões mínimas necessárias para acessar um repositório específico dentro de uma organização no GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] Adicione o usuário como colaborador externo ao repositório com acesso `Read`, permitindo que ele clone o repositório sem permitir que faça push de alterações.
- [ ] Conceda automaticamente a eles acesso como 'Owner' para garantir que possam gerenciar as configurações da organização e o acesso ao repositório.
- [ ] Forneça a eles acesso 'Write' a todos os repositórios da organização para garantir que possam contribuir para os projetos.
- [ ] Não é possível conceder acesso a repositórios específicos; é necessário dar acesso à organização inteira.
