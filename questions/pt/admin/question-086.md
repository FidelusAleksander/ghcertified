---
question: "Como adicionar um runner self-hosted a um repositório do GitHub?"
documentation: "https://docs.github.com/en/actions/hosting-your-own-runners/managing-self-hosted-runners/adding-self-hosted-runners"
---

- [x] Navegue até as configurações do repositório, selecione Actions, depois Runners, e siga as instruções para baixar, configurar e executar o aplicativo de runner self-hosted.
- [ ] Clone diretamente o repositório para sua máquina local e execute um script específico para adicioná-lo automaticamente como um runner self-hosted.
- [ ] Envie um e-mail para o suporte do GitHub solicitando a adição de um runner self-hosted ao seu repositório.
- [ ] Use a API do GitHub para enviar uma solicitação para adicionar um runner self-hosted sem nenhuma configuração manual.
