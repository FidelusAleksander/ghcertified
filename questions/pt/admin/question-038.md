question: "Como o proprietário de uma organização pode encontrar estatísticas sobre o uso de licenças para seu GitHub Enterprise Server?"
documentation: "https://docs.github.com/en/enterprise-server@3.21/billing/how-tos/manage-plan-and-licenses/view-enterprise-usage"

- [x] Navegando até o painel `Site admin` para visualizar o uso de licenças nas configurações da `Enterprise account`.
- [ ] Através de solicitações por e-mail diretamente ao suporte do GitHub.
- [ ] Verificando a página de perfil público da organização para estatísticas em tempo real.
- [ ] Usando a API do GitHub para buscar dados de uso de licenças em tempo real para o GitHub Enterprise Server.
