---
question: "O que é necessário para que um usuário se autentique com uma conta de Enterprise Managed User?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/identity-and-access-management/understanding-iam-for-enterprises/about-enterprise-managed-users#authenticating-with-a-managed-user-account"
---

- [x] Os usuários devem se autenticar no IdP da empresa para acessar os recursos no GitHub.com.
- [ ] Os usuários se autenticam por meio de uma página de login pública do GitHub sem qualquer interferência da empresa.
- [ ] A autenticação é ignorada para Enterprise Managed Users para facilitar o acesso.
- [ ] Os usuários fornecem um token de autenticação especial gerenciado pelo GitHub.
