---
question: "Quando uma lista de permissões de IP do Enterprise está em vigor, o que acontece quando você tenta publicar seu site do GitHub Pages a partir de um branch em vez de usar um workflow personalizado do GitHub Actions?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/configuration/hardening-security-for-your-enterprise/restricting-network-traffic-to-your-enterprise-with-an-ip-allow-list#using-github-pages-with-an-ip-allow-list"
---

- [x] O runner de build terá acesso ao repositório para o site do GitHub Pages por padrão
- [ ] O runner de build será bloqueado de acessar o repositório até ser adicionado manualmente à lista de permissões
- [ ] O GitHub Pages não pode ser criado ou atualizado até que a lista de permissões de IP seja desativada
- [ ] Você deve acionar manualmente os builds para sites do GitHub Pages
