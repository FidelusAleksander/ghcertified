---
question: "Quais são os principais recursos e funcionalidades das equipes dentro de uma organização do GitHub?"
documentation: "https://docs.github.com/en/organizations/organizing-members-into-teams/about-teams"
---

- [x] As equipes em organizações do GitHub permitem gerenciar o acesso e as notificações para grupos que refletem a estrutura da empresa, com capacidades de acesso administrativo, leitura ou escrita aos repositories, menções para notificações e o uso de equipes aninhadas para gestão hierárquica.
- [ ] As equipes são apenas para networking social dentro do GitHub, permitindo que os membros de uma organização se comuniquem e compartilhem atualizações sem nenhum impacto no acesso aos repositories ou nas notificações organizacionais.
- [ ] As equipes do GitHub são idênticas aos membros da organização, sem recursos ou funcionalidades distintas, como gestão de acessos ou notificações.
- [ ] As equipes dentro do GitHub podem incluir outside collaborators e são usadas principalmente para comunicação externa, sem qualquer estrutura organizacional interna ou permissões de acesso aos repositories.
