---
question: "Qual afirmação é verdadeira sobre nomes de usuário e informações de perfil para Enterprise Managed Users?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/concepts/identity-and-access-management/enterprise-managed-users"
---

- [x] Os nomes de usuário e informações de perfil são configurados pelo IdP da empresa e não podem ser alterados pelos usuários.
- [ ] Os usuários podem alterar seus nomes de usuário e informações de perfil a qualquer momento no GitHub.
- [ ] Os nomes de usuário e informações de perfil são definidos pelos próprios usuários durante a configuração da conta do GitHub.
- [ ] O GitHub atribui automaticamente nomes de usuário e informações de perfil aleatórios para maior segurança.
