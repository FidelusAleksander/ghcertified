---
question: "O que é necessário para baixar ou publicar um GitHub Package em workflows, como com GitHub Actions ou outras ferramentas de CI/CD?"
documentation: "https://docs.github.com/en/packages/learn-github-packages/introduction-to-github-packages#authenticating-to-github-packages"
---

- [x] Use `GITHUB_TOKEN` para publicar pacotes associados ao repositório do workflow, e um PAT para instalar pacotes de repositórios privados.
- [ ] GitHub Packages não pode ser usado em workflows do GitHub Actions ou em nenhuma ferramenta de CI/CD.
- [ ] Apenas chaves SSH são necessárias para usar o GitHub Packages em workflows.
- [ ] É necessário realizar o download e upload manual de pacotes para cada execução do workflow.
