---
question: "Qual é a diferença entre GitHub Apps e OAuth apps?"
documentation: "https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/differences-between-github-apps-and-oauth-apps"
---

- [x] GitHub Apps oferecem permissões granulares para repositórios específicos, enquanto OAuth Apps solicitam acesso aos dados do usuário em todos os repositórios.
- [ ] GitHub Apps e OAuth Apps não diferem; eles oferecem o mesmo nível de permissões.
- [ ] OAuth Apps oferecem permissões mais granulares em comparação com GitHub Apps.
- [ ] GitHub Apps só podem acessar repositórios públicos, enquanto OAuth Apps podem acessar repositórios públicos e privados.
