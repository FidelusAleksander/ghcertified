---
question: "Qual é o custo de uso do GitHub Actions para repositórios públicos?"
documentation: "https://docs.github.com/en/billing/concepts/product-billing/github-actions"
---

- [x] O uso do GitHub Actions é gratuito para repositórios públicos.
- [ ] O GitHub Actions cobra por cada minuto utilizado em repositórios públicos.
- [ ] Repositórios públicos recebem metade da quantidade de minutos gratuitos em comparação com repositórios privados.
- [ ] O GitHub Actions não está disponível para repositórios públicos.
