---
question: "Como o GitHub Copilot lida com a retenção de dados para sugestões de código no IDE?"
documentation: "https://resources.github.com/learn/pathways/copilot/essentials/how-github-copilot-handles-data/"
---

- [x] As sugestões são mantidas temporariamente na memória e descartadas após o uso, não sendo gravadas no disco
- [ ] Todas as sugestões são permanentemente armazenadas em um banco de dados local para referência futura
- [ ] As sugestões são automaticamente salvas nos repositórios do GitHub para controle de versão
- [ ] Trechos de código são armazenados em cache no disco por 30 dias antes de serem excluídos
