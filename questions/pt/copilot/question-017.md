---
question: "Como o GitHub Copilot lida com a retenção de dados para sugestões de código no IDE?"
documentation: "https://learn.github.com/learning-pathways/github-copilot"
---

- [x] As sugestões são mantidas temporariamente na memória e descartadas após o uso, sem serem gravadas no disco
- [ ] Todas as sugestões são armazenadas permanentemente em um banco de dados local para referência futura
- [ ] As sugestões são automaticamente salvas em repositórios do GitHub para controle de versão
- [ ] Trechos de código são armazenados em cache no disco por 30 dias antes de serem excluídos
