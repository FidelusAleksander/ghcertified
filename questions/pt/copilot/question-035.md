---
question: "Você está desenvolvendo um aplicativo em Kotlin. O que você deve levar em consideração ao usar o GitHub Copilot?"
documentation: "https://learn.microsoft.com/en-us/training/modules/github-copilot-across-environments/2-code-completion-with-git-hub-copilot"
---

- [x] Kotlin não está incluído nos idiomas onde o GitHub Copilot oferece suporte avançado. Como resultado, as sugestões de código podem não ter a mesma qualidade que as sugestões em um idioma com melhor suporte, como Ruby, Java ou C#.
- [ ] Kotlin não está incluído nos idiomas onde o GitHub Copilot oferece suporte avançado. Como resultado, o GitHub Copilot não será capaz de explicar ou corrigir o código.
- [ ] Kotlin não está incluído nos idiomas onde o GitHub Copilot oferece suporte avançado. Como resultado, o GitHub Copilot ainda poderá explicar ou corrigir o código, mas não será capaz de fornecer sugestões de código.
- [ ] Nenhuma das opções está correta, pois Kotlin está incluído nos idiomas onde o GitHub Copilot oferece suporte avançado.
