---
question: "Quais afirmações estão corretas sobre o GitHub Copilot CLI?"
documentation: "https://learn.microsoft.com/en-us/training/modules/github-copilot-across-environments/4-git-hub-copilot-for-the-command-line"
---

- [x] Os prompts não são armazenados, enquanto as análises de uso são mantidas por padrão.
- [x] É possível desativar as análises de uso opcionais.
- [x] As respostas geradas pelo Copilot CLI podem ser revisadas após cada sugestão.
- [ ] Tanto os prompts quanto as análises de uso são armazenados por padrão.
> Os prompts não são armazenados por padrão.
- [ ] Não é possível desativar as análises de uso opcionais porque são anonimizadas.
- [ ] O GitHub Copilot CLI somente pode explicar ou sugerir comandos, mas não executá-los em nome do usuário.
