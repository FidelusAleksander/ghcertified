---
question: "Para que é usado um alias ao utilizar o GitHub Copilot?"
documentation: "https://learn.microsoft.com/en-us/training/modules/github-copilot-across-environments/4-git-hub-copilot-for-the-command-line"
---

- [x] Usar um alias permite que o GitHub Copilot CLI execute automaticamente comandos na linha de comando.
- [ ] Usar um alias permite indicar nos commits do Git que o GitHub Copilot foi o autor das alterações.
- [ ] Usar um alias permite que o GitHub revise e crie pull requests em seu nome.
