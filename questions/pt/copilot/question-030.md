---
question: "O que são smart actions?"
documentation: "https://learn.microsoft.com/en-us/training/modules/get-started-github-copilot/11-examine-inline-chat-smart-actions-quick-chat"
---

- [x] Smart actions são tarefas comuns e predefinidas, como explicar código, corrigir código ou gerar testes e documentos, que podem ser realizadas pelo GitHub Copilot Chat sem a necessidade de redigir um prompt para elas.
- [ ] Smart actions é uma modalidade avançada do GitHub Copilot usada para depuração e correção de testes com falha. 
- [ ] Smart actions são tarefas predefinidas projetadas para ajudar com tarefas comuns de codificação dentro de uma organização, que podem ser configuradas no nível do repositório e da organização.
