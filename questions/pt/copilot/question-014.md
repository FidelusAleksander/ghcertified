---
question: "Qual é a maneira mais fácil de começar a usar o GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [ ] Solicitar acesso ao suporte do GitHub e esperar pela aprovação antes de usar o GitHub Copilot.
- [ ] Usar o site do Copilot e colar seu código ao pedir sugestões.
- [x] Instalar a extensão do Copilot no ambiente de sua preferência, como o Visual Studio Code.
- [ ] Criar um novo repositório público no GitHub e habilitar o Copilot para analisar seu código e fornecer sugestões.
