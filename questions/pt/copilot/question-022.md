---
question: "Qual é o principal objetivo do comando '/tests' do GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/tutorials/write-tests"
---

- [x] Ele gera um conjunto de testes unitários para o arquivo atualmente aberto, usando o contexto de arquivos de teste existentes, se disponível
- [ ] Ele executa todos os testes unitários existentes no projeto sem gerar novos
- [ ] Ele apenas valida a sintaxe de arquivos de teste existentes sem criar novos testes
- [ ] Ele remove permanentemente todos os arquivos de teste existentes para começar do zero
