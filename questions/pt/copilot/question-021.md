---
question: "Qual afirmação descreve corretamente o GitHub Copilot CLI?"
documentation: "https://docs.github.com/en/copilot/concepts/agents/copilot-cli/about-copilot-cli"
---

- [x] Ele permite que você use o Copilot a partir do seu terminal para responder perguntas, escrever e depurar código, e interagir com o GitHub.com
- [ ] Ele é limitado à geração de aliases de shell para `gh copilot suggest` e `gh copilot explain`
- [ ] Ele funciona apenas dentro da interface web do GitHub e não pode acessar arquivos de projetos locais
- [ ] Ele executa automaticamente todos os comandos de shell sugeridos sem pedir aprovação
