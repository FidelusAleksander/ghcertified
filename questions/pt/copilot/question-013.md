- [x] O preenchimento de código não estará disponível nos arquivos afetados.
- [x] O conteúdo nos arquivos afetados não influenciará as sugestões de preenchimento de código em outros arquivos.
- [ ] O conteúdo nos arquivos afetados continuará a influenciar as respostas do GitHub Copilot Chat.
- [ ] O preenchimento de código não será afetado nos arquivos afetados.
