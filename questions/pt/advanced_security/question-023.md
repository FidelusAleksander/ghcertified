---
question: "Quais informações os alertas do Dependabot fornecem?"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/dependabot-alerts/about-dependabot-alerts"
---

- [x] Os alertas do Dependabot informam que seu repositório utiliza um pacote inseguro.
- [ ] Os alertas do Dependabot informam que seu repositório está sendo usado por outros repositórios públicos.
- [ ] Os alertas do Dependabot informam que seu repositório utiliza uma versão não testada de um pacote.
- [ ] Os alertas do Dependabot informam que seu repositório utiliza uma versão desatualizada de um pacote.
> O Dependabot faz isso com atualizações de versão, mas não com alertas.
