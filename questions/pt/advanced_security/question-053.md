---
question: "Como você pode ativar a análise de código para um repositório?"
documentation: "https://docs.github.com/en/code-security/how-tos/find-and-fix-code-vulnerabilities/configure-code-scanning/configure-code-scanning"
---

- [x] Vá para a guia de segurança nas configurações do repositório e ative a análise de código com a configuração padrão ou avançada.
- [ ] Vá para as configurações do usuário e ative a análise de código; você pode optar por habilitá-la para todos ou apenas para repositórios selecionados.
- [ ] Adicione um arquivo de configuração `.github/codeql.yml` ao repositório.
- [ ] Vá para a guia de segurança nas configurações do repositório e responda a um questionário sobre o conteúdo do repositório. Com base nas respostas, o GitHub ativará a análise de código com a configuração apropriada.
