---
question: "Como você pode ativar a análise de código para um repositório?"
documentation: "https://docs.github.com/en/code-security/code-scanning/enabling-code-scanning/configuring-default-setup-for-code-scanning"
---

- [x] Vá para a aba de segurança nas configurações do repositório e ative a análise de código com configuração padrão ou avançada.
- [ ] Vá para as configurações de usuário e ative a análise de código; você pode optar por ativá-la para todos os repositórios ou apenas para repositórios selecionados.
- [ ] Adicione um arquivo de configuração `.github/codeql.yml` ao repositório.
- [ ] Vá para a aba de segurança nas configurações do repositório e responda a um questionário sobre o conteúdo do repositório. Com base nas respostas, o GitHub ativará a análise de código com a configuração apropriada.
