- [x] A varredura de segredos verifica seu repositório em busca de segredos, como chaves privadas ou tokens.
- [ ] A varredura de segredos verifica seu repositório em busca de possíveis vulnerabilidades de código que possam expor segredos, como chaves privadas ou tokens.
- [ ] A varredura de segredos é uma ferramenta para armazenamento e gerenciamento seguro de segredos.
- [ ] A varredura de segredos é um git hook que verifica seus commits em busca de segredos, como chaves privadas ou tokens, antes de serem enviados para o GitHub.
