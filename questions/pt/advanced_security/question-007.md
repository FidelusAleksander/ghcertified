- [x] A verificação de segredos analisa seu repositório em busca de segredos como chaves privadas ou tokens.
- [ ] A verificação de segredos analisa seu repositório em busca de possíveis vulnerabilidades de código que possam expor segredos como chaves privadas ou tokens.
- [ ] A verificação de segredos é uma ferramenta para armazenamento e gerenciamento seguro de segredos.
- [ ] A verificação de segredos é um gancho do Git que analisa seus commits em busca de segredos como chaves privadas ou tokens antes de serem enviados para o GitHub.
