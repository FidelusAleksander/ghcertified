question: "O que são os Avisos de Segurança de Repositório?"
documentation: "https://docs.github.com/en/code-security/concepts/vulnerability-reporting-and-management/repository-security-advisories"
---

- [x] Um espaço privado onde os mantenedores do repositório podem discutir vulnerabilidades e questões de segurança dentro da base de código.
- [ ] Especialistas em segurança do GitHub que ajudam os usuários do GitHub Enterprise com seus problemas de segurança.
- [ ] Uma lista de problemas de segurança que estão disponíveis publicamente para que qualquer pessoa possa ver e evitar.
- [ ] Um lugar para reunir e discutir publicamente questões de segurança na comunidade de código aberto.
