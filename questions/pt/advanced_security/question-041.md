---
question: "Como o CodeQL pode ser utilizado em um sistema de CI externo juntamente com repositórios GitHub?"
documentation: "https://docs.github.com/en/code-security/how-tos/find-and-fix-code-vulnerabilities/integrate-with-existing-tools/use-with-existing-ci-system#about-using-code-scanning-with-your-existing-ci-system"
---

- [x] Executar o CodeQL CLI no sistema de CI externo para analisar o código e enviar os resultados para o repositório GitHub
- [ ] O CodeQL não pode ser usado em sistemas de CI externos; ele é exclusivo para o GitHub Actions
- [ ] Enviar o código-fonte para o GitHub para análise e, em seguida, baixar os resultados para uso no sistema de CI
- [ ] Executar manualmente o CodeQL localmente e enviar os resultados por e-mail para os administradores do repositório GitHub
