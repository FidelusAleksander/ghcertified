---
question: "Como o CodeQL pode ser usado em um sistema de CI externo juntamente com repositórios GitHub?"
documentation: https://docs.github.com/en/code-security/how-tos/find-and-fix-code-vulnerabilities/integrate-with-existing-tools/use-with-existing-ci-system#about-using-code-scanning-with-your-existing-ci-system
---

- [x] Execute o CodeQL CLI no sistema de CI externo para verificar o código e enviar os resultados para o repositório GitHub
- [ ] CodeQL não pode ser usado em sistemas de CI externos; ele é exclusivo do GitHub Actions
- [ ] Envie o código-fonte para o GitHub para análise e depois baixe os resultados para uso no sistema de CI
- [ ] Execute manualmente o CodeQL localmente e envie os resultados por e-mail para os administradores do repositório GitHub
