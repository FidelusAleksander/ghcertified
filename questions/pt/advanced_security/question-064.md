documentation: "https://codeql.github.com/docs/writing-codeql-queries/about-codeql-queries/"
---

- [x] As consultas CodeQL podem ser executadas contra um banco de dados CodeQL para identificar padrões que podem indicar erros de codificação ou vulnerabilidades de segurança.
- [ ] As consultas CodeQL analisam sua base de código e são usadas para criar um banco de dados CodeQL.
- [ ] As consultas CodeQL são usadas para fins de revisão de código no GitHub.
- [ ] As consultas CodeQL são perguntas baseadas em texto que você pode fazer ao mecanismo CodeQL sobre sua base de código.
