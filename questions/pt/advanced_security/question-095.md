---
question: "Como você pode definir uma política de segurança padrão para todos os repositórios na Organização GitHub `my-org`?"
documentation: "https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file#supported-file-types"
---

- [x] Criando um arquivo `SECURITY.md` no repositório `my-org/.github`
- [ ] Editando a política de segurança nas configurações de `Code Security and analysis` da organização
- [ ] Políticas de segurança padrão só podem ser definidas pelo suporte do GitHub
- [ ] Você pode definir uma política de segurança padrão para todos os repositórios na Organização GitHub `my-org` adicionando um arquivo `SECURITY.md` a cada repositório individualmente.
