question: "Como você pode definir uma política padrão de relatório de vulnerabilidades de segurança para todos os repositórios na Organização GitHub `my-org`?"

- [x] Criando um arquivo `SECURITY.md` no repositório `my-org/.github`
- [ ] Configurando uma política de relatório de vulnerabilidades de segurança nas configurações de `Code Security and analysis` da organização
- [ ] Políticas de segurança padrão só podem ser definidas pelo suporte do GitHub
- [ ] Criando um arquivo `SECURITY.md` em cada repositório individual
