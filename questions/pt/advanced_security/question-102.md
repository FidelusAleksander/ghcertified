---
question: "Quem pode ativar os alertas do Dependabot em um repositório?"
documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-alerts/about-dependabot-alerts#configuration-of-dependabot-alerts"
---

- [x] Proprietários do repositório e pessoas com acesso de administrador
- [ ] Apenas o proprietário do repositório 
- [ ] Os alertas do Dependabot são ativados em todos os repositórios pelo GitHub e não podem ser desativados ou ativados por nenhum indivíduo.
- [ ] Os alertas do Dependabot são ativados adicionando uma GitHub Action ao repositório, portanto, qualquer pessoa com acesso de gravação ao repositório pode ativá-los.
