---
question: "Ao usar o GitHub Actions como seu sistema de CI e uma ferramenta de terceiros para executar a verificação de código, como você pode enviar os resultados SARIF para o GitHub?"
documentation: "https://docs.github.com/en/code-security/code-scanning/integrating-with-code-scanning/uploading-a-sarif-file-to-github#uploading-a-code-scanning-analysis-with-github-actions"
---

- [x] Usando o `github/codeql-action/upload-sarif` GitHub Action
- [ ] Quando se utiliza o GitHub Actions, os resultados SARIF são enviados automaticamente para o GitHub.
- [ ] Você só pode usar CodeQL ao executar a verificação de código no GitHub Actions. Ferramentas de terceiros para verificação de código não são suportadas.
- [ ] Usando o `actions/upload-artifact` GitHub Action
