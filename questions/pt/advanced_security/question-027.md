---
question: "Quando o gráfico de dependências do GitHub para o seu repositório será atualizado?"
documentation: "https://docs.github.com/en/code-security/supply-chain-security/understanding-your-software-supply-chain/about-supply-chain-security#what-is-the-dependency-graph"
---

- [x] Quando alguém faz um push de uma alteração para o repositório de uma de suas dependências.
- [x] Quando você faz um push de um commit para o branch padrão do repositório, apenas se isso alterar ou adicionar um arquivo de manifesto/trava suportado.
- [ ] Quando você faz um push de qualquer commit para o branch padrão do repositório.
- [ ] Quando o seu repositório publica uma nova versão.
- [ ] Quando o seu repositório publica um novo tag git.
- [ ] Quando o fluxo de trabalho do GitHub Actions que usa a ação do GitHub `actions/dependency-graph` é acionado.
