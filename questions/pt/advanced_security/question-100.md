question: "O que são atualizações de segurança do Dependabot?"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/dependabot-security-updates/about-dependabot-security-updates"
---

- [x] É uma funcionalidade do Dependabot que cria automaticamente pull requests para atualizar dependências vulneráveis no seu repositório.
- [ ] É uma funcionalidade do Dependabot que cria uma lista de dependências vulneráveis no seu repositório.
- [ ] É uma funcionalidade do Dependabot que cria alertas quando uma vulnerabilidade de segurança é detectada em uma de suas dependências.
> Isso se refere aos alertas do Dependabot; as atualizações de segurança do Dependabot são baseadas nos alertas do Dependabot.
- [ ] É uma funcionalidade do Dependabot que cria automaticamente pull requests para atualizar dependências no seu repositório quando elas lançam uma nova versão.
> Isso se refere às atualizações de versão do Dependabot.
