---
question: "O que são atualizações de segurança do Dependabot?"
documentation: "https://docs.github.com/en/code-security/how-tos/secure-your-supply-chain/secure-your-dependencies/configure-security-updates"
---

- [x] É um recurso do Dependabot que cria automaticamente pull requests para atualizar dependências vulneráveis no seu repositório.
- [ ] É um recurso do Dependabot que cria uma lista de dependências vulneráveis no seu repositório.
- [ ] É um recurso do Dependabot que cria alertas quando uma vulnerabilidade de segurança é detectada em uma das suas dependências.
> Isso é Dependabot alerts, as atualizações de segurança do Dependabot são baseadas nos alertas do Dependabot.
- [ ] É um recurso do Dependabot que cria automaticamente pull requests para atualizar dependências no seu repositório quando uma nova versão é lançada.
> Isso é Dependabot version updates
