---
question: "Qual é o propósito do programa de parceiros de verificação de segredos?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/secret-scanning-partner-program"
---

- [x] Provedores de serviço podem se tornar parceiros do GitHub para que o formato de seus segredos seja reconhecido pela verificação de segredos do GitHub.
- [ ] O programa de parceiros do GitHub permite que empresas e organizações com licença do GitHub Advanced Security usem a verificação de segredos do GitHub para escanear seus repositórios.
- [ ] O GitHub faz parceria com empresas de segurança externas para fornecer verificação de segredos para repositórios do GitHub.
- [ ] É um programa onde profissionais de segurança registrados podem, de boa fé, reportar ao GitHub quaisquer segredos que encontrarem em repositórios do GitHub e receber recompensas por isso.
