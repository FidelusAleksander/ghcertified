---
question: "Qual é o objetivo do programa de parceria de varredura de segredos?"
documentation: "https://docs.github.com/en/code-security/tutorials/secret-scanning-partner-program"
---

- [x] Provedores de serviço podem fazer parceria com o GitHub para que o formato de seus segredos seja reconhecido pela varredura de segredos do GitHub.
- [ ] O programa de parceria do GitHub permite que empresas e organizações com licença do GitHub Advanced Security usem a varredura de segredos do GitHub para escanear seus repositórios.
- [ ] O GitHub faz parceria com empresas de segurança externas para fornecer varredura de segredos para repositórios do GitHub.
- [ ] É um programa onde profissionais de segurança cadastrados podem, de boa fé, relatar ao GitHub quaisquer segredos que encontrarem em repositórios do GitHub e receber recompensas por isso.
