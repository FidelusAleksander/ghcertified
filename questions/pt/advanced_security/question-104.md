- [x] Acesse as configurações de `Code security and analysis` da organização e ative os alertas do Dependabot para todos os repositórios de uma vez.
- [ ] Tornar todos os repositórios na organização privados.
- [ ] Em todos os repositórios da organização, execute a GitHub Action `actions/enable-ghas` com o parâmetro `alerts` definido como `true`.
- [ ] Criar um script que ativará os alertas do Dependabot em todos os repositórios da organização.
