question: "O que é uma política de segurança do GitHub?"
documentation: "https://docs.github.com/en/code-security/getting-started/adding-a-security-policy-to-your-repository#about-security-policies"
---

- [x] É um documento que instrui os usuários sobre como relatar de forma responsável vulnerabilidades de segurança em um projeto. Ele é normalmente definido em um arquivo `SECURITY.md` em um repositório.
- [ ] É uma ferramenta para corrigir automaticamente vulnerabilidades de segurança no seu código.
- [ ] É um recurso que permite criptografar seu repositório.
- [ ] Uma política de segurança do GitHub é um serviço de assinatura que oferece proteção antivírus para seus projetos.
