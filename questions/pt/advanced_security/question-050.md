- [x] 500 para organizações/enterprises e 100 para repositórios.
- [ ] 100 para organizações/enterprises e 500 para repositórios.
- [ ] 100 para organizações, enterprises e repositórios.
- [ ] Não há limite para o número de padrões personalizados que você pode definir para a verificação de segredos no GitHub.
