---
question: "Qual é a diferença entre GitHub projects e GitHub projects classic?"
documentation: "https://docs.github.com/en/issues/organizing-your-work-with-project-boards/managing-project-boards/about-project-boards"
---

- [x] GitHub Projects é a nova experiência do GitHub que oferece muitas novas funcionalidades e melhorias em relação ao antigo GitHub Projects classic. 
- [ ] GitHub Projects classic é a nova experiência do GitHub que oferece muitas novas funcionalidades e melhorias em relação ao antigo GitHub Projects.
- [ ] GitHub Projects classic é uma maneira mais simples de gerenciar seus GitHub Projects e é recomendado para iniciantes.
- [ ] GitHub Projects classic é a versão aprimorada que, além do GitHub Projects, oferece alguns templates pré-definidos, como roteiros e sprints.
