---
question: "Qual é a diferença entre GitHub Projects e GitHub Projects classic?"
documentation: "https://docs.github.com/en/issues/planning-and-tracking-with-projects"
---

- [x] GitHub Projects é a nova experiência do GitHub que oferece muitos novos recursos e melhorias em relação ao antigo GitHub Projects classic. 
- [ ] GitHub Projects classic é a nova experiência do GitHub que oferece muitos novos recursos e melhorias em relação ao antigo GitHub Projects.
- [ ] GitHub Projects classic é uma maneira mais simples de gerenciar seus GitHub Projects e é recomendado para iniciantes.
- [ ] GitHub Projects classic são as versões aprimoradas que, além do GitHub Projects, oferecem alguns modelos pré-configurados, como roteiros e sprints.
