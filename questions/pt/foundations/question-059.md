---
question: "O que é InnerSource?"
documentation: "https://resources.github.com/innersource/what-is-innersource/"
---

- [x] InnerSource refere-se à prática de aplicar princípios de open source dentro de uma organização
- [ ] InnerSource refere-se à prática de aplicar princípios de open source em projetos pessoais
- [ ] InnerSource refere-se à base de conhecimento interna de uma organização sobre o projeto em que estão trabalhando.
- [ ] InnerSource é uma ferramenta que permite às organizações ver quais projetos open source estão utilizando em seu código.
