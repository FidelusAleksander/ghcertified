---
question: "O que é InnerSource?"
documentation: "https://github.com/resources/articles/innersource"
---

- [x] InnerSource refere-se à prática de aplicar princípios de código aberto dentro de uma organização
- [ ] InnerSource refere-se à prática de aplicar princípios de código aberto em projetos pessoais
- [ ] InnerSource refere-se à base de conhecimento interna de uma organização sobre o projeto em que estão trabalhando.
- [ ] InnerSource é uma ferramenta que permite às organizações ver quais projetos de código aberto estão usando em seu código.
