question: "O que é um git commit?"
documentation: "https://docs.github.com/en/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/about-commits"
---

- [x] Um commit é uma captura de um repositório em um momento específico.
- [ ] Um commit refere-se a um arquivo novo ou atualizado em um repositório.
- [ ] Um commit refere-se às alterações de código feitas em um repositório em um pull request.
- [ ] Um commit é um texto legível que descreve as alterações feitas em um repositório.
