---
question: "Quais são os comandos de barra do GitHub?"
documentation: "https://docs.github.com/en/issues/tracking-your-work-with-issues/using-issues/about-slash-commands"
---

- [x] É uma forma de inserir rapidamente Markdown complexo em seus comentários e descrições de pull request ou issue.
- [ ] É uma forma de automatizar GitHub Actions.
- [ ] É outro nome para o GitHub CLI.
- [ ] É uma maneira de corrigir rapidamente problemas de formatação de código nas alterações de código do seu pull request.
