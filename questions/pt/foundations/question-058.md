---
question: "O que é o GitHub Marketplace?"
documentation: "https://docs.github.com/en/apps/github-marketplace/github-marketplace-overview/about-github-marketplace-for-apps"
---

- [x] GitHub Marketplace é um lugar onde os desenvolvedores podem encontrar ferramentas que ajudam a construir software diretamente junto com seu código.
- [ ] GitHub Marketplace é um lugar onde as empresas podem encontrar desenvolvedores para contratar.
- [ ] GitHub Marketplace é um lugar onde os desenvolvedores oferecem a venda de propriedade de seus repositórios.
- [ ] GitHub Marketplace é um lugar onde você pode patrocinar desenvolvedores de código aberto com base em uma assinatura.
> Isso é o GitHub Sponsors.
