- [x] É um aplicativo GUI para trabalhar com Git e GitHub no seu computador.
- [ ] É uma versão auto-hospedada do GitHub que você pode instalar em seus próprios servidores ou computador pessoal.
- [ ] É um plano de preços do GitHub para contas pessoais que oferece recursos adicionais além do plano gratuito.
- [ ] É um editor online que permite trabalhar no seu repositório diretamente no navegador.
