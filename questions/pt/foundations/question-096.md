---
question: "Qual dessas afirmações sobre as diferenças entre issues e discussions é verdadeira?"
documentation: "https://docs.github.com/en/discussions/quickstart"
---

- [x] GitHub Discussions são para conversas que precisam ser transparentes e acessíveis, mas não precisam ser rastreadas em um quadro de projeto e não estão relacionadas ao código, ao contrário das GitHub Issues.
- [ ] GitHub Issues são para conversas que precisam ser transparentes e acessíveis, mas não precisam ser rastreadas em um quadro de projeto e não estão relacionadas ao código, ao contrário das GitHub Discussions.
