- [x] É um grupo de usuários do GitHub da mesma organização.
- [ ] É uma versão do Microsoft Teams para o GitHub.
- [ ] É uma forma de gerenciar o acesso a repositórios pessoais do GitHub para muitos usuários de uma só vez.
> Repositórios pessoais não podem ser compartilhados com equipes, apenas com colaboradores individuais.
