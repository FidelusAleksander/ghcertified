- [ ] Eles podem excluir todas as execuções de workflows de repositórios privados com mais de 90 dias.  
- [ ] Eles podem desativar as revisões de código para repositórios na organização.  
- [ ] Eles podem remover a regra de proteção de branch `Require status checks to pass before merging`.
