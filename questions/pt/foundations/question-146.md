---
question: "Qual é a diferença entre Git e GitHub?"
documentation: "https://learn.microsoft.com/en-us/training/modules/intro-to-git/1-what-is-vc"
---

- [x] O Git permite que você trabalhe com um ou mais branches locais e envie alterações para um repositório remoto. O GitHub atua como o repositório remoto, acessado por meio de um site ou ferramentas de linha de comando.
- [ ] O Git é um sistema de controle de versão distribuído (DVCS) que funciona na nuvem. O GitHub é uma camada de interface que fornece acesso à tecnologia Git.
- [ ] O Git é usado por um colaborador individual. O GitHub é usado por vários colaboradores para simplificar o trabalho de desenvolvimento em grupo.
