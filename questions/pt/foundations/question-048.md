question: "O que são Repository Insights?"

- [x] É um local onde você pode ver dados sobre o repositório, como frequência de código, lista dos colaboradores mais ativos e mais. 
- [ ] Repository Insights é um recurso que permite ver os segredos do repositório e os alertas de varredura de código.
- [ ] É uma lista de boas práticas a seguir ao administrar um repositório.
- [ ] É um recurso que permite ver a visão geral de segurança do repositório.
