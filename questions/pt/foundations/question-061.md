---
question: "O que é um fork no GitHub?"
documentation: "https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/about-forks"
---

- [x] Um fork é uma cópia pessoal de um repositório de outro usuário que vive na sua conta.
- [ ] Um fork é o lugar onde um branch se divide em pelo menos dois outros branches.
- [ ] Um fork é o estado de um repositório quando ele não está atualizado com o repositório remoto.
- [ ] Um fork é um branch que não está atualizado com o branch padrão do repositório.
