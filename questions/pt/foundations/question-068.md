- [x] GitHub é uma plataforma que utiliza o Git como sua tecnologia central e adiciona recursos que facilitam a colaboração e o gerenciamento de código. 
- [ ] GitHub é um sistema de controle de versão distribuído com recursos que facilitam a colaboração em projetos.
- [ ] GitHub é uma plataforma de compartilhamento de arquivos com recursos que facilitam a colaboração em projetos usando controle de versão centralizado.
- [ ] GitHub é uma plataforma moderna de CI/CD com recursos que ajudam a impulsionar o desenvolvimento rápido de produtos.
