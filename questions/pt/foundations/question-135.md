- [ ] O GitHub Pages permite hospedar um site privado gratuitamente, acessível apenas por membros da organização.  
> Os sites do GitHub Pages são sempre acessíveis publicamente, a menos que você utilize um site de projeto com regras de firewall empresarial no GitHub Enterprise Server.  
- [x] O GitHub Pages permite hospedar sites estáticos a partir de um repositório GitHub.  
> **Correto**. O GitHub Pages é especificamente para hospedagem de sites estáticos a partir do seu repositório.  
- [ ] O GitHub Pages converte automaticamente arquivos Java em aplicativos web dinâmicos.  
- [ ] O GitHub Pages só pode ser usado por contas de organizações.  
> Tanto contas pessoais quanto de organizações podem usar o Pages.
