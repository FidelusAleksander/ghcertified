---
question: "Quais são as diferenças entre GitHub Copilot Individual e GitHub Copilot Business?"
documentation: "https://docs.github.com/en/copilot/about-github-copilot/subscription-plans-for-github-copilot"
---

- [x] GitHub Copilot Individual só pode ser usado por contas pessoais e GitHub Copilot Business é usado por organizações e empresas
- [ ] GitHub Copilot Individual usa um modelo de IA treinado em código-fonte público e GitHub Copilot Business usa um modelo de IA treinado em código-fonte privado
- [ ] GitHub Copilot Individual é gratuito e GitHub Copilot Business é um recurso pago
- [ ] GitHub Copilot Individual só pode ser usado ao trabalhar em repositórios públicos e GitHub Copilot Business pode ser usado ao trabalhar em repositórios públicos e privados dentro de uma organização
