---
question: "Como você pode começar a usar o GitHub Copilot após ativar a assinatura do GitHub Copilot?"
documentation: "https://docs.github.com/en/copilot/get-started/quickstart"
---

- [x] Configure o GitHub Copilot em um dos IDEs suportados, como Visual Studio Code ou JetBrains, e comece a codificar
- [ ] O GitHub Copilot começará automaticamente a fazer sugestões em pull requests e issues no seu repositório
- [ ] Você precisa configurar uma GitHub Action que irá configurar o GitHub Copilot no seu repositório
- [ ] Você precisa editar as configurações do repositório e ativar o GitHub Copilot para o repositório
