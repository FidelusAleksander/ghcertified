- [ ] Você pode fixar até 10 repositórios no seu perfil.  
- [x] Você pode fixar até seis repositórios ou gists no seu perfil para acesso rápido.  
> **Correto**. O GitHub permite fixar até seis itens (repositórios ou gists).  
- [ ] Você só pode fixar repositórios públicos, não privados.  
- [ ] Fixar um item o remove da lista de repositórios.
