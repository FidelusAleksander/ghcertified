question: "O que são Enterprise Managed Users?"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/admin/identity-and-access-management/understanding-iam-for-enterprises/about-enterprise-managed-users#about-enterprise-managed-users"
---

- [x] É um recurso do GitHub Enterprise que permite gerenciar o acesso dos usuários a partir de um provedor de identidade externo, como Azure AD ou Okta
- [ ] É um usuário de suporte do GitHub Enterprise que pode ser usado para gerenciar sua Enterprise pelo Suporte do GitHub mediante solicitação
- [ ] É uma conta de usuário compartilhada que pode ser utilizada por vários usuários na sua Enterprise
- [ ] É uma conta de usuário sem acesso à interface web do GitHub, usada apenas para tarefas de automação no seu GitHub Enterprise
