---
question: "O que é um plano GitHub Pro?"
documentation: "https://docs.github.com/en/get-started/learning-about-github/githubs-plans#github-pro"
---

- [x] GitHub Pro é um plano pago para contas pessoais que oferece recursos adicionais em comparação ao plano gratuito.
- [ ] GitHub Pro é um plano pago para organizações que oferece recursos adicionais em comparação ao plano gratuito para organizações.
- [ ] GitHub Pro é um caminho de certificação após o qual você se torna um GitHub Pro.
- [ ] GitHub Pro é um plano pago para empresas hospedarem o GitHub em seus próprios servidores.
