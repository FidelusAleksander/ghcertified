- [x] É um ambiente de desenvolvimento pré-configurado especificamente configurado para um repositório. Ele permite que você comece a escrever código para um projeto imediatamente, sem precisar configurar um ambiente de desenvolvimento local.
- [ ] Um GitHub Codespace é um ambiente de codificação holográfico interativo que requer hardware especial para ser usado.
- [ ] O GitHub Codespace é uma ferramenta de codificação com inteligência artificial que gera código quase perfeito automaticamente.
- [ ] O GitHub Codespace é um evento anual hospedado pelo GitHub, onde os desenvolvedores podem se encontrar e discutir tópicos de codificação.
