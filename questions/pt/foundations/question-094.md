---
question: "O que são pull requests de rascunho?"
documentation: "https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests#draft-pull-requests"
---

- [x] Pull requests de rascunho são pull requests que ainda não estão prontos para serem revisados.
- [ ] Pull requests de rascunho são pull requests que já foram revisados, mas não foram mesclados.
- [ ] Pull requests de rascunho são pull requests que já foram revisados, mas não foram aprovados.
- [ ] Pull requests de rascunho são templates para criar novos pull requests.
