question: "Como você pode atribuir uma pessoa a uma issue ou pull request?"
documentation: "https://docs.github.com/en/issues/tracking-your-work-with-issues/using-issues/assigning-issues-and-pull-requests-to-other-github-users"

- [x] Usando o campo Assignees na barra lateral
- [ ] Adicionando uma etiqueta com o nome de usuário do GitHub deles à issue ou pull request
- [ ] Mencionando eles na descrição da issue ou pull request
- [ ] Não é possível atribuir uma pessoa a uma issue ou pull request.
