question: "Como você pode substituir um GitHub Secret de nível organizacional `API_KEY` com um valor diferente ao trabalhar dentro de um repositório?"
documentation: "https://docs.github.com/en/actions/reference/security/secrets"
---

- [x] Criando um secret do repositório com o mesmo nome `API_KEY`
- [x] Criando um secret de ambiente com o mesmo nome `API_KEY`
- [ ] Criando um secret da empresa com o mesmo nome `API_KEY`
- [ ] Criando um secret da empresa com o nome `OVERRIDE_API_KEY`
- [ ] Criando um secret do repositório com o nome `OVERRIDE_API_KEY`
- [ ] Criando um secret de ambiente com o nome `OVERRIDE_API_KEY`
- [ ] Criando um secret do repositório com o nome `REPOSITORY_API_KEY`
- [ ] Criando um secret de ambiente com o nome `ENVIRONMENT_API_KEY`
