question: "Como as organizações que estão utilizando o GitHub Enterprise Server podem habilitar a sincronização automática de GitHub Actions de terceiros hospedadas no GitHub.com para sua instância do GitHub Enterprise Server?"

- [x] Usando o GitHub Connect
- [ ] O GitHub Enterprise Server tem acesso a todas as GitHub.com Actions por padrão
> GitHub Actions no GitHub Enterprise Server é projetado para funcionar em ambientes sem acesso completo à internet. Por padrão, workflows não podem usar ações do GitHub.com e do GitHub Marketplace.
- [ ] Usando a ferramenta actions-sync
> https://docs.github.com/en/enterprise-server@3.6/admin/github-actions/managing-access-to-actions-from-githubcom/manually-syncing-actions-from-githubcom#about-the-actions-sync-tool
- [ ] O GitHub Enterprise Server não pode usar ações do GitHub.com devido à sua natureza on-premise e falta de acesso à internet
