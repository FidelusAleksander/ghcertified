---
question: "O que é verdadeiro sobre variáveis padrão?"
documentation: "https://docs.github.com/en/actions/reference/workflows-and-actions/variables"
---

- [x] Variáveis de ambiente padrão são definidas pelo GitHub e não especificadas em um workflow
- [x] A maioria das variáveis de ambiente padrão possui uma propriedade de contexto correspondente
- [x] Atualmente, o valor da variável de ambiente padrão de CI pode ser sobrescrito, mas não é garantido que isso sempre será possível
- [ ] Você pode adicionar uma nova variável de ambiente padrão adicionando o prefixo “GITHUB_” a ela
- [ ] Variáveis de ambiente padrão sempre têm o prefixo “GITHUB_”
- [ ] Variáveis de ambiente padrão podem ser acessadas usando o contexto env
