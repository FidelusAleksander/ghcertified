---
question: "Quais componentes podem ser reutilizados dentro de uma GitHub Organization?"
---

- [x] Secrets
- [x] Configuration Variables
- [x] Self Hosted Runners
- [x] Workflow Templates
- [ ] Artifacts
> Os artifacts são usados para preservar dados após a conclusão de um job e/ou compartilhar esses dados com outro job dentro do mesmo workflow.
- [ ] Cache
> O cache pode ser reutilizado entre workflows dentro de um único repository.
- [ ] Environment Variables
> As environment variables podem ser definidas para um passo, job ou workflow. Elas não podem ser compartilhadas entre workflows/repositories ou organizations.
