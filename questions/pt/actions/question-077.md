---
question: "O que é um runner auto-hospedado?"
documentation: "https://docs.github.com/en/actions/hosting-your-own-runners/managing-self-hosted-runners/about-self-hosted-runners"
---

- [x] Um runner auto-hospedado é um sistema que você implanta e gerencia para executar trabalhos do GitHub Actions no GitHub.com
- [ ] Um runner auto-hospedado é um sistema para enviar código para um servidor privado
- [ ] Um runner auto-hospedado é um sistema para criar cargas de trabalho automaticamente
- [ ] Um runner auto-hospedado é um sistema para gerenciar pull requests dos usuários da organização
