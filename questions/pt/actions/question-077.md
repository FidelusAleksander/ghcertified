---
question: "O que é um self-hosted runner?"
documentation: "https://docs.github.com/en/actions/concepts/runners/self-hosted-runners"
---

- [x] Um self-hosted runner é um sistema que você implementa e gerencia para executar jobs do GitHub Actions no GitHub.com
- [ ] Um self-hosted runner é um sistema para enviar código para um servidor privado
- [ ] Um self-hosted runner é um sistema para ser capaz de criar cargas de trabalho automaticamente
- [ ] Um self-hosted runner é um sistema para gerenciar pull requests de usuários da organização
