- [x] Nível de Job
- [x] Nível de Step
- [ ] Nível de Workflow
> `if:` não está presente no nível de workflow. Para acionar condicionalmente um workflow inteiro, outros métodos devem ser usados, como especificar tipos de atividade.
- [ ] Nível de Ambiente
- [ ] Nível de Organização
