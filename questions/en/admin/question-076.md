---
question: "What's the difference between GitHub Apps and OAuth apps?"
documentation: "https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/differences-between-github-apps-and-oauth-apps"
---

- [x] GitHub Apps offer granular permissions to specific repositories, while OAuth Apps request access to user data across all repositories.
- [ ] GitHub Apps and OAuth Apps do not differ; they provide the same level of permissions.
- [ ] OAuth Apps offer more granular permissions compared to GitHub Apps.
- [ ] GitHub Apps can only access public repositories, whereas OAuth Apps can access both public and private repositories.
