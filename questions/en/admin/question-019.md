- [x] Users must authenticate on the company's IdP to access resources on GitHub.com.
- [ ] Users authenticate through a public GitHub login page without any enterprise interference.
- [ ] Authentication is bypassed for enterprise managed users for ease of access.
- [ ] Users provide a special GitHub-managed authentication token.
