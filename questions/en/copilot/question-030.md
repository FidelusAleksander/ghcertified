question: "What are smart actions?"
documentation: "https://learn.microsoft.com/en-us/training/modules/get-started-github-copilot/11-examine-inline-chat-smart-actions-quick-chat"
---

- [x] Smart actions are common and predefined tasks such as explaining code, fixing code or generating tests and docs that can be performed by GitHub Copilot Chat without the need to draft a prompt for them.
- [ ] Smart actions is an advanced GitHub Copilot modality used for debugging and fixing failing tests. 
- [ ] Smart actions are predefined tasks designed to assist with common coding tasks inside an organization, which can be set on repository and organization level.
