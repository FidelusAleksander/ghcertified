---
question: "Which of the GitHub Copilot subscription plans let you specify coding guidelines for code reviews?"
documentation: "https://learn.microsoft.com/en-us/training/modules/get-started-github-copilot/3-compare-github-copilot-plans"
---

- [x] Copilot Enterprise
> Coding guidelines for code reviews require a custom instructions file in the `.github` repository at the organization level, which is an Enterprise-only feature.
- [ ] Copilot Business and Copilot Enterprise
> Copilot Business supports code review but not custom coding guidelines.
- [ ] Copilot Pro+, Copilot Business and Copilot Enterprise
- [ ] Copilot Pro, Copilot Pro+, Copilot Business and Copilot Enterprise
