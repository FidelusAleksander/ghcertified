question: "What is the purpose of the `codeql database analyze` command in CodeQL CLI?"
documentation: "https://docs.github.com/en/code-security/codeql-cli/getting-started-with-the-codeql-cli/analyzing-your-code-with-codeql-queries#running-codeql-database-analyze"

- [x] Analyzing a CodeQL database, producing results usually in the form of a SARIF file.
- [ ] Analyzing a CodeQL database, producing results usually in the form of security advisories.
- [ ] Analyzing the source code, producing a CodeQL database.
- [ ] Analyzing a CodeQL database and uploading the results to GitHub.
> The results are not uploaded to GitHub, a separate command is used for that.
