- [x] It's a feature that allows Dependabot to automatically dismiss Dependabot alerts that match certain criteria.
- [ ] Auto-triage rules are defined in the `dependabot.yml` configuration file to specify which package managers should be used to scan your project for vulnerabilities.
- [ ] Dependabot auto-triage rules are used for automatically deleting old dependencies in your project.
- [ ] Auto triage rules define how often Dependabot should scan your project for vulnerabilities.
