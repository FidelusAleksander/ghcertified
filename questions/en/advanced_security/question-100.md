question: "What are Dependabot security updates?"
documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-security-updates/about-dependabot-security-updates"
---

- [x] It's a Dependabot feature that automatically creates pull requests to update vulnerable dependencies in your repository.
- [ ] It's a Dependabot feature that creates a list of vulnerable dependencies in your repository.
- [ ] It's a Dependabot feature that creates alerts when a security vulnerability is detected in one of your dependencies.
> That's Dependabot alerts, Dependabot security updates are based on Dependabot alerts.
- [ ] It's a Dependabot feature that automatically creates pull requests to update dependencies in your repository when they release a new version.
> That's Dependabot version updates
