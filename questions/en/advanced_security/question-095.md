- [x] By creating a `SECURITY.md` file in the `my-org/.github` repository
- [ ] By editing the security policy in the organization's `Code Security and analysis` settings
- [ ] Default security policies can only be set by GitHub support
- [ ] You can set a default security policy for all repositories in `my-org` GitHub Organization by adding a `SECURITY.md` file to each individual repository.
