---
question: "What information do Dependabot alerts provide?"
documentation: "https://docs.github.com/en/code-security/dependabot/dependabot-alerts/about-dependabot-alerts"
---

- [x] Dependabot alerts tell you that your repository uses a package that is insecure.
- [ ] Dependabot alerts tell you that your repository is being used by other public repositories.
- [ ] Dependabot alerts tell you that your repository uses an untested version of a package.
- [ ] Dependabot alerts tell you that your repository uses an outdated version of a package
> Dependabot does that with version updates, but not with alerts.
