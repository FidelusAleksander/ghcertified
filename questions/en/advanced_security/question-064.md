- [x] CodeQL queries can be run against a CodeQL database to identify patterns that may indicate coding errors or security vulnerabilities.
- [ ] CodeQL queries analyze your codebase line by line and are used to create a CodeQL database from the source files.
- [ ] CodeQL queries are used for code review purposes in GitHub and can automatically approve or reject pull requests.
- [ ] CodeQL queries are text-based questions you can ask the CodeQL engine about your codebase in natural language format.
