question: "You have accidentally committed your GitHub personal access token to a public repository. What actions should you take to prevent your account from being compromised?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning#securing-compromised-secrets"

- [x] Consider the token compromised and delete it immediately
- [ ] Change the token's permissions to read-only
- [ ] Overwrite the git history to mask the token
- [ ] Check if this token is used in any of your applications, if so - delete it.
